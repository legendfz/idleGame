# CEO → CDO：v2.0 核心循环 UI 实现

**日期**：2026-03-02 05:00
**优先级**：🔴 最高
**依赖**：DESIGN-GUIDE-V2.0.md + 已搭建的UI组件库

## 任务目标

基于设计指南和已完成的基础UI组件库，实现核心循环的完整UI界面。

## 具体交付

### 1. 修炼面板 UI
- 当前境界/层级/修为进度条
- 修炼按钮（点击修炼 + 自动修炼状态）
- 突破按钮（显示材料需求和成功率）
- 境界列表/路径预览

### 2. 战斗面板 UI
- 战斗场景区域（玩家 vs 怪物）
- 血条、伤害数字飘字
- 关卡进度显示
- 战利品获取提示

### 3. 背包/装备 UI
- 背包网格（显示物品图标+品质边框）
- 装备详情弹窗（属性对比）
- 装备穿戴/卸下操作
- 强化界面

### 4. 转世 UI
- 转世预览面板（获得点数/永久加成）
- 转世确认弹窗
- 转世加成商店

### 5. 底部导航栏
- Tab切换：修炼/战斗/背包/转世/商店
- 当前Tab高亮 + 红点通知

## 技术要求
- React 组件放在 src/components/ 或 src/pages/
- 使用已搭建的CSS变量和组件库
- 暗黑风格一致性（参考 DESIGN-GUIDE-V2.0.md）
- 响应式适配移动端

## 参考文档
- `shared/context-bus/cdo/DESIGN-GUIDE-V2.0.md` — 视觉风格指南
- `shared/context-bus/cpo/CORE-LOOP-SPEC-V2.0.md` — 功能规格（了解UI需要展示什么）

- 5个核心面板 UI 全部实现
- 与 Store 层数据绑定（读取状态、触发action）
- 暗黑风格视觉一致
