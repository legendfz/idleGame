# v1.1 UI 增量设计交付

## 交付物
→ `CDO/UI-DESIGN-V1.1.md`

## 内容摘要

### 1. 六级品质配色与特效
- 完整 6 级体系：凡品(灰) / 良品(绿) / 上品(蓝) / 极品(紫) / 仙品(橙) / 神品(红)
- 每级定义：主色、衬色、前缀符号、字重、text-shadow
- 掉落特效分 6 级递进：无 → slide-in → 边缘闪光 → 全屏闪屏+Toast
- 神品专属：边框红金脉动、「天降神兵！」逐字出现
- 提供全部 CSS @keyframes 代码

### 2. 离线收益展示页面
- 三区结构：基础收益（灵石/经验/蟠桃）→ 战斗成果（击杀/升级）→ 装备掉落
- 5 阶段入场动画序列：弹窗弹入 → 数字 countUp → 战斗成果淡入 → 装备逐行 slide-in → 按钮出现
- 边界情况处理：<1min 不弹、>24h 封顶、无掉落隐藏区块

### 3. 反馈入口
- 位置：更多 Tab 菜单列表
- 4 种反馈类型：🐛Bug / 💡建议 / ⚖️数值 / 💬其他
- 自动收集设备/游戏上下文
- MVP 方案：Google Form 无后端提交
