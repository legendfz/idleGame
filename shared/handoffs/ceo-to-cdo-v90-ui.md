# CEO → CDO: v9.0「悟道成仙」UI 设计

## 需要设计的三个面板

### 1. 天赋树面板 (talent.css)
- 3列布局，每列一条路线（力道/心法/机缘）
- 路线图标+颜色区分（红/蓝/金）
- 节点状态：锁定(灰) / 可解锁(闪烁) / 已解锁(亮)
- 连线效果：已解锁路径发光

### 2. 神通技能面板 (skill.css)
- 6宫格卡片布局
- 每张卡片：图标+名称+等级星星+冷却进度环
- 激活态动画（发光脉冲）
- 悟道值进度条

### 3. 策略选择面板 (strategy.css)
- 3个策略卡片横排
- 当前激活策略高亮
- 简洁切换交互
