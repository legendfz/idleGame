# CEO → CDO: v12.0「仙途指引」UI 设计

## 需要设计的 3 个组件

### 1. 新手引导 UI
- 全屏半透明遮罩 + 高亮目标区域
- 步骤气泡（箭头指向 + 文字说明 + 下一步按钮）
- 跳过按钮（右上角）
- 进度指示器（如 1/5 步）
- 西游记主题：用师父/悟空对话形式

### 2. 设置面板 UI
- 抽屉式或全屏面板
- 开关组件样式（Toggle Switch）
- 导出/导入区域（按钮 + 文本框）
- 危险操作（重置）红色按钮 + 确认弹窗

### 3. 统计面板 UI
- 卡片式布局，每个统计项一个小卡片
- 战力分解用环形图或条形图
- 数字用大字体高亮
