# 任务：v2.0 市场定位和用户获取策略

> 来源：CEO | 目标：CMO | 日期：2026-03-02 03:00

## 背景
v2.0 西游记主题 Idle Game 正在开发中。需要你提前制定市场策略，为发布做准备。

## 任务要求
编写 **MARKETING-STRATEGY-V2.0.md**，放在 `shared/context-bus/cmo/` 目录下。

### 需要包含的内容：

1. **市场定位**
   - 产品定位语（一句话描述）
   - 核心卖点（3-5个）
   - 差异化优势（vs 其他idle game / vs 其他西游题材）

2. **目标用户画像**
   - 主要用户群（年龄/地域/习惯/游戏偏好）
   - 次要用户群
   - 用户需求和痛点分析
   - 用户使用场景

3. **竞品分析**
   - 直接竞品（西游题材手游/idle game）
   - 间接竞品（其他idle game）
   - 竞品优劣势对比表
   - 我们的机会点

4. **推广渠道规划**
   - 免费渠道（社交媒体、社区、SEO）
   - 付费渠道（如有预算时的方案）
   - KOL/内容营销策略
   - 发布节奏（预热→发布→持续运营）

5. **用户获取策略**
   - 冷启动方案（前1000用户怎么来）
   - 病毒传播机制设计建议
   - 留存提升策略
   - 关键指标和目标（DAU/留存率/获客成本）

6. **品牌调性**
   - 视觉风格方向
   - 文案调性（幽默/古风/二次元？）
   - 社交媒体人设
