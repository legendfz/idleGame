# v1.2 Sprint 1 — 精炼系统 & 高阶强化 UI 设计

## 交付物
→ `CDO/UI-V12-REFINE-ENHANCE.md`

## 内容摘要

### 一、精炼系统 UI
- **入口**：装备详情弹窗，仅 +10 满级装备显示 `[ 🔥 精炼 ]`
- **界面**：精炼对象 + 星级显示（☆★ 5阶）+ 所需材料 + 效果预览 + 成功率
- **确认弹窗**：3阶及以上弹出风险提示（降级警告红色加粗）
- **碎片进度条**：材料 Tab 中显示碎片收集进度，满碎片自动提示合成
- **动画**：
  - 成功 4 阶段（聚能→聚爆→升星弹入→结果面板）
  - 失败无降级（熄灭→单线框结果）
  - 失败降级（红闪+星碎裂→双线框震动）

### 二、鸿蒙高阶强化 UI（+11~+15）
- **入口**：极品★及以上 +10 装备显示 `[ ⚡ 高阶强化 ]` 渐变金色按钮
- **界面**：强化详情 + 消耗 + 成功率 + 风险区 + 保护道具（可折叠，复选框选用）
- **保护道具**：防降符（防降级）、幸运符（+10%成功率），选中后风险区/成功率实时更新
- **确认弹窗**：无保护时弹出降级风险二次确认（红色危险按钮）
- **动画**：
  - 成功（闪电聚能→雷击→金框结果，比普通强化更华丽）
  - 失败降级（短路→红闪→shake+countDown 红色递减）
  - 失败有保护（弱闪→蓝色盾牌闪烁→等级不变）
- **+15 特殊**：渐变金红色等级数字 + ⚡ 标记 + 背包行渐变边框
