# CEO → CPO: 武器锻造系统 + 材料获取 + Boss难度曲线 PRD

**优先级**: P0 — 立即执行
**日期**: 2026-03-02

## 任务

更新 PRD，新增以下三大系统设计：

### 1. 武器锻造系统
- 玩家收集矿石/灵材/Boss掉落等材料
- 通过锻造台合成/升级武器
- 不同材料组合产出不同属性武器（攻击/防御/暴击/吸血等）
- 高级武器需要稀有材料
- 设计锻造配方表（材料组合 → 武器产出）
- 武器品质等级与现有品质体系对齐（凡/灵/仙/神/混沌/鸿蒙）
- 锻造成功率/失败惩罚机制

### 2. Boss 难度递增设计
- 后期 Boss 不会被轻易击杀
- 需要特定装备/等级/策略才能通关
- Boss 特殊机制设计：
  - 免疫（特定伤害类型免疫）
  - 反击（受击时反伤）
  - 多阶段（血量阈值触发形态变化）
  - 狂暴（时间限制，超时后Boss增强）
- Boss 难度曲线公式（与玩家战力的关系）

### 3. 多元材料获取玩法
- **采集系统**：矿脉/药田/灵泉，不同地点产出不同材料，有采集CD
- **探索副本/秘境**：限时挑战，产出稀有材料，每日次数限制
- **交易/炼化**：低级材料合成高级材料，转化比例设计
- **每日任务/成就奖励**：任务列表+奖励表
- **挂机产出**：不同地点挂机产出不同材料，产出速率设计
