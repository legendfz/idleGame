# v21.0「音画仙境」— 视觉+音效+动画升级

## 背景
游戏功能已非常丰富（20个版本迭代），但视觉体验仍是纯文字+emoji。
本版本目标：让游戏**看起来像游戏**，提升留存和第一印象。

## 三大方向

### 1. CSS视觉重做（CDO负责）
- 全局配色方案：深色仙侠风（渐变背景、发光效果）
- 卡片式UI：装备/成就/角色用卡片+阴影+圆角
- 品质颜色统一：凡(白)/灵(绿)/仙(蓝)/神(紫)/混沌(橙)/鸿蒙(金+发光)
- 战斗区域：HP条动画、伤害数字弹出、暴击放大效果
- 底部Tab栏美化：选中态高亮+小动画

### 2. CSS动画系统（CTO负责）
- 战斗攻击动画（shake/flash）
- 装备获得动画（从天降下+发光）
- 升级/突破全屏特效（金光闪烁）
- 数值变化动画（数字滚动）
- Tab切换过渡动画（slide）
- 用纯CSS实现，不引入额外动画库

### 3. 音效系统（CTO负责）
- Web Audio API 合成音效（不用音频文件）
- 攻击命中音、暴击音、升级音、装备获得音、按钮点击音
- 音效开关（集成到设置面板已有的动画开关旁边）
- 音量控制滑块

## 技术约束
- 纯CSS动画+Web Audio API，零外部依赖
- bundle增加不超过5KB gzip
- 所有动画尊重设置面板的"动画开关"
- 音效模块独立，方便后续扩展
