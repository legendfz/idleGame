# v1.2 Sprint 2 — UI 规范交付

## 交付物
→ `CDO/UI-V12-SPRINT2.md`

## 内容摘要

### 1. 离线保底提示 UI
- 与现有离线弹窗整合，在战斗成果区后插入保底奖励区
- **100关保底**：单线框 + 绿色（灵品级）+ slide-up 动画
- **500关保底**：双线框 + 橙色（仙品级）+ 全屏橙闪 + 更长延迟入场
- 多次触发合并显示，边界情况完整覆盖
- 数据接口定义（`OfflineReport` 扩展 + `PityReward`）

### 2. 背包分解确认 UI
- 背包满时弹出提示 → 进入分解专属页面
- **智能推荐**：自动选中低品质/重复装备，高品质/套装件不推荐
- **批量操作**：全选推荐 / 全选 / 清除
- **分解预览**：实时计算灵石+精炼碎片+精铁产出
- **二次确认**：含高品质装备时额外蓝色警告
- 分解公式定义供 CTO 参考

### 3. 反馈模板选择 UI
- 入口：模板卡片列表（图标+标题+描述+箭头）
- **🐛 Bug 报告**：位置单选 + 问题描述 + 复现步骤 + 自动上下文
- **💡 功能建议**：类型单选 + 标题 + 详细描述
- **💬 体验反馈**：5 星评分 + 多选改进方面 + 文字反馈
- 通用表单组件 CSS（input/textarea/checkbox/radio/submit）
- 提交状态流程（loading → 成功 Toast / 失败重试）
