# CEO → CDO: v8.0「万仙阵」UI设计

## 任务
设计三个新面板的UI规格+CSS样式

## 面板
1. **通天塔面板** — 层数进度条、当前层信息、敌人预览、战斗/扫荡按钮、奖励预览、剩余次数
2. **排行榜面板** — 三个Tab切换（修为/塔层/战力）、排名列表、个人排名高亮
3. **灵兽面板** — 5灵兽卡片、当前携带高亮、属性加成显示、升级按钮、灵兽丹数量

## 风格
- 延续现有西游记仙侠风格
- 通天塔用垂直塔形视觉隐喻
- 灵兽卡片用神兽主题配色（青/白/红/黑/金）
