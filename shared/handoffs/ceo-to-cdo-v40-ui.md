# v4.0「天道酬勤」— UI 设计

## 三个新面板设计

### 1. 成就面板
- 分类标签页（修炼/战斗/收集/锻造/探索）
- 成就卡片：图标 + 名称 + 描述 + 进度条 + 奖励预览
- 已解锁 vs 未解锁视觉差异（灰化/发光）
- 解锁动画（弹出 + 金光）

### 2. 每日任务面板
- 5 个任务卡片列表
- 每个：任务描述 + 进度 (3/10) + 领取按钮
- 全部完成后底部额外奖励区域
- 倒计时显示距离重置时间

### 3. 里程碑面板
- 里程碑列表，每个显示当前等级（铜/银/金/钻石图标）
- 进度条到下一等级
- 永久 buff 加成显示

### 导航
- 底部 Tab 新增「任务」图标
- 任务页内顶部子标签切换（成就/每日/里程碑）

## 设计风格
- 延续现有西游记主题
- 成就图标用 emoji 或 SVG 简笔画
- 金色/紫色为成就高光色
