# PRD v13.0 —「洞天福地」

**版本**：13.0 | **作者**：CPO | **日期**：2026-03-03
**状态**：Draft → 待 CEO 审批
**依赖**：PRD-V2.0 ~ V12.0

---

## 1. 系统概览

```
洞天（个人仙府）→ 建筑产出 → 被动资源+加成
秘境探索 → 随机事件链 → 稀有材料+图纸+仙缘碎片
仙缘（NPC好感） → 赠礼/任务/偶遇 → 永久buff+专属神通
```

三系统联动：秘境产出建筑图纸加速洞天发展；秘境偶遇提升仙缘好感；仙缘 buff 反哺秘境战斗与洞天产出。

## 3. 秘境探索

### 3.1 基本规则

| 属性 | 值 |
|------|-----|
| 每日免费次数 | 3 |
| 额外次数购买 | 最多 +2 次（30/60 灵石） |
| 每次节点数 | 5~8（随机） |
| 秘境持续 | 即时完成（非实时） |

### 3.2 难度等级

| 难度 | emoji | 战力要求 | 节点数 | 奖励倍率 |
|------|-------|---------|--------|---------|
| 凡境 | 🟢 | 无 | 5 | ×1 |
| 灵境 | 🔵 | 10,000 | 6 | ×2 |
| 仙境 | 🟣 | 50,000 | 7 | ×4 |
| 神境 | 🔴 | 200,000 | 8 | ×8 |

### 3.3 节点类型

| 类型 | emoji | 概率 | 效果 |
|------|-------|------|------|
| ⚔️ 战斗 | ⚔️ | 35% | 遭遇敌人，胜利获奖励，失败秘境结束 |
| 📦 宝箱 | 📦 | 25% | 随机材料/金币/灵石 |
| 🌟 机缘 | 🌟 | 15% | 偶遇仙缘NPC（好感+5~15） |
| ⚡ 陷阱 | ⚡ | 15% | 损失 10~20% 当前HP，或消耗材料 |
| 🏪 商人 | 🏪 | 10% | 可用金币购买稀有材料/蓝图（折扣价） |

### 3.4 战斗节点

- 敌人属性 = 秘境难度基础 × (1 + 节点序号 × 0.15)
- 基础属性：凡境 = 玩家 ×0.6, 灵境 = ×0.8, 仙境 = ×1.0, 神境 = ×1.3
- 战斗失败：秘境立即结束，已获奖励保留
- 战斗胜利：获得材料+经验+金币

## 5. Store 设计

### 5.1 sanctumStore（洞天）

```typescript
interface SanctumState {
  buildings: {
    spiritField: number;   // 灵田 1-10
    pillRoom: number;      // 丹房 1-10
    library: number;       // 藏经阁 1-10
    forgeFurnace: number;  // 锻造炉 1-10
    spiritArray: number;   // 聚灵阵 1-10
  };
  pendingRewards: {
    spiritStones: number;
    pills: number;
  };
  lastTickTime: number;    // 上次结算时间戳
  blueprints: number;      // 蓝图数量
}
```

### 5.2 explorationStore（秘境）

```typescript
interface ExplorationState {
  dailyAttempts: number;
  dailyPurchased: number;
  lastResetDate: string;
  currentRun: {
    active: boolean;
    difficulty: 'mortal' | 'spirit' | 'immortal' | 'divine';
    totalNodes: number;
    currentNode: number;
    nodeLog: { type: string; result: string; rewards: string }[];
    collectedRewards: Record<string, number>;
  } | null;
  explorationTokens: number;  // 探索令
}
```

### 5.3 affinityStore（仙缘）

```typescript
interface AffinityNpc {
  id: string;
  name: string;
  emoji: string;
  affinity: number;       // 0-100
  dailyGiftsGold: number; // 今日金币赠礼次数
  dailyGiftsGem: number;  // 今日灵石赠礼次数
  dailyQuestDone: boolean;
  skillUnlocked: boolean;
  skillCooldownUntil: number;
}

interface AffinityState {
  npcs: Record<string, AffinityNpc>;
  affinityFragments: number;  // 仙缘碎片
  lastResetDate: string;
}
```

### 5.4 存档集成

`getFullState()` / `loadFullState()` 必须包含 sanctumStore + explorationStore + affinityStore。

## 7. 经济影响

| 来源 | 产出/日（预估中期） |
|------|-------------------|
| 灵田 Lv.5 | 灵石 ~180 (3/min × 60min 在线) |
| 丹房 Lv.5 | 小修为丹 ~30 |
| 秘境(3次灵境) | 灵石 ~15, 金币 ~30,000 |

**评估**：
- 灵田灵石 +180/日偏高（现有中期~100/日），**建议**灵田产出降至 0.3/min Lv1, +0.3/级 → Lv10 = 3.0/min → 在线1h约180，日活跃2h约360
- 实际需根据日均在线时长调整，建议 CTO 实现时加入产出上限（每日灵田最多产 100 灵石）
- 秘境灵石可控（每日 ~15），不破坏平衡 ✅
- 丹房产小修为丹为资源转化，不直接产灵石 ✅
