## 1. 系统概览

```
通关81难+圣人境 → 积累功德值 → 选择轮回道 → 重生+道果加成
                                                  ↓
秘境深渊(1-100层) ← 战力提升 ← 道果+装备+天赋
                                                  ↓
排行榜(NPC对比) → 每日排名奖励 → 资源加速循环
```

## 2. 六道轮回系统（Prestige 2.0）

### 2.1 功德值（新核心资源）

**获取途径**：

| 来源 | 数量 | 频率 |
|------|------|------|
| 击败章节 Boss（每 9 难）| 10 × 章节数 | 一次性 |
| 击败第 81 难 | 500 | 一次性/轮 |
| 每日任务全勤 | 15 | 每日 |
| 秘境深渊（每 10 层）| 5 × 层级段 | 一次性 |
| 成就解锁 | 5~50 不等 | 一次性 |
| 连续登录（7 日）| 30 | 每周 |

**首次轮回预估**：
- 9 章 Boss：10+20+30+40+50+60+70+80+90 = 450
- 第 81 难：500
- 每日任务（~7 天）：105
- 成就（预估）：~150
- **总计 ≈ 1,200 功德（约 7 天可达首次轮回门槛 1,000）**

### 2.4 轮回流程

```
1. 条件检查: 通关81难 + 圣人境 + 功德≥门槛
2. 选择道: 展示六道面板 + 预览加成
3. 确认: 扣除功德，记录道次
4. 重置: 境界→凡人, 修为→0, 金币→0, 装备→清空, 进度→第1难
5. 保留: 功德余额, 佛缘, 天赋点(+3), 伙伴, 道果, 成就, 里程碑, 秘境最高层
6. 加成生效: 本轮道加成 + 所有道果永久加成
7. 快速重获: 剧情法宝第1难自动给
```

### 2.5 加成递减机制

同一道连续轮回，加成递减：

```
effective_bonus = base_bonus × diminishing(count)
diminishing(n) = 1 / (1 + 0.1 × (n - 1))
// 第1次: 100%, 第2次: 91%, 第5次: 71%, 第10次: 53%
```

设计意图：鼓励轮回不同道以收集全部道果，而非单刷一道。

## 3. 秘境深渊（Deep Dungeon）

### 3.1 基本结构

- **总层数**：100 层
- **分段**：每 10 层一个 Boss 层，每 25 层一个大 Boss
- **每层内容**：3 波小怪 + 1 精英/Boss
- **进度保存**：可随时退出，下次从当前层继续

### 3.2 层数难度公式

```
floor_hp(n) = floor(5000 × 1.5^(n-1) × tier_bonus(n))
floor_atk(n) = floor(50 × 1.4^(n-1) × tier_bonus(n))

tier_bonus(n):
  n % 10 == 0 (Boss层):    × 3.0
  n % 25 == 0 (大Boss):    × 5.0 (覆盖10层bonus)
  n % 5 == 0 (精英层):     × 1.5
  else:                     × 1.0
```

## 4. 排行榜系统（本地模拟）

### 4.1 排行榜类型

| 排行榜 | 排名依据 | NPC 数量 |
|--------|---------|---------|
| 境界榜 | 境界order×9+subLevel | 50 |
| 秘境榜 | 最高通关层数 | 50 |
| 战力榜 | 综合战力值 | 50 |

### 4.2 NPC 生成公式

```
npc_power(rank) = player_estimated_peak × decay(rank)

decay(rank):
  rank 1: × 1.2  (略强于预估)
  rank 2-5: × (1.1 - rank×0.02)
  rank 6-20: × (0.9 - (rank-6)×0.03)
  rank 21-50: × (0.5 - (rank-21)×0.01)

npc_name: 随机西游主题名 (「散仙·青云」「妖王·赤虎」等)
npc_realm: 根据 power 反算境界
```

NPC 数据每次轮回时重新生成（匹配玩家当前实力）。

### 4.3 排名奖励（每日结算）

| 排名 | 金币 | 灵石 | 材料 | 功德 |
|------|------|------|------|------|
| 1 | 50K | 30 | T3×3 | 10 |
| 2-3 | 30K | 20 | T3×2 | 7 |
| 4-10 | 15K | 10 | T2×3 | 5 |
| 11-20 | 8K | 5 | T2×2 | 3 |
| 21-50 | 3K | 2 | T1×3 | 1 |

### 4.4 战力计算公式

```
combat_power = (attack + defense×0.5 + hp×0.01) × realm_mult
             × (1 + crit_rate×0.01 × crit_dmg)
             × (1 + equip_score×0.001)
             × (1 + talent_score×0.01)
             × (1 + pet_score×0.01)

equip_score = Σ(装备品质分 × (1 + 强化级×0.1))
talent_score = 已分配天赋点 × 2
pet_score = 活跃伙伴等级 × 3
```
