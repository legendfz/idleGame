## 1. 系统概览

```
采集/副本/任务 → 材料积累 → 锻造台合成 → 高品质武器 → 挑战高难 Boss
        ↑                                                    ↓
        └──────────── Boss 掉落稀有材料 ←─────────────────────┘
```

三大子系统形成正循环：**材料获取 → 锻造装备 → 挑战更强 Boss → 获得更稀有材料**。

## 3. Boss 难度递增设计

### 3.1 Boss 属性公式

```
boss_hp(n) = floor(500 × 2.2^(n-1) × tier_bonus(n) × phase_bonus)
boss_atk(n) = floor(15 × 1.8^(n-1) × tier_bonus(n))
boss_def(n) = floor(5 × 1.6^(n-1) × tier_bonus(n))

tier_bonus(n):
  n % 9 == 0 (章节Boss): × 3.0
  n % 3 == 0 (小节点):   × 1.5
  else:                   × 1.0

phase_bonus: 多阶段 Boss 后续阶段额外倍率
```

### 3.3 Boss 特殊机制详解

#### 免疫机制
```
immunity(boss, damage_type) → 0 伤害
触发条件：特定 Boss 对特定伤害类型免疫
持续：周期性开启（如每 15 秒免疫物理 3 秒）
破解：切换伤害类型 / 等待免疫结束 / 使用破甲道具
```

| Boss | 免疫类型 | 持续 | CD | 破解方式 |
|------|---------|------|-----|---------|
| 黄风大王(18) | 物理 | 3s | 15s | 法术攻击 / 等待 |
| 红孩儿(36) | 火属性 | 持续 | — | 观音法水(消耗品) |
| 大鹏(54) | 近战 | 5s | 20s | 远程攻击 / 法宝 |

#### 反击机制
```
counter_damage = boss_atk × counter_rate
触发条件：玩家攻击时一定概率触发
counter_rate: 10%~30%（随难度递增）
```

| Boss | 反击率 | 反击伤害 | 应对 |
|------|-------|---------|------|
| 黑熊精(3) | 10% | ATK×0.5 | 堆HP |
| 牛魔王(45) | 20% | ATK×0.8 | 吸血装备 |
| 九头虫(72) | 25% | ATK×1.0 | 免伤+吸血 |
| 老龟(81) | 30% | ATK×1.2 | 全减伤套 |

#### 多阶段机制
```
phase_trigger: HP < threshold%
effect: 属性倍率提升 + 新技能解锁
```

| Boss | 阶段 | 触发阈值 | 效果 |
|------|------|---------|------|
| 寅将军王(9) | P2 | 60% HP | ATK×1.3 |

| | P3 | 33% HP | 3分身 + 本体隐身5s |
| 牛魔王(45) | P2 | 50% HP | 变身巨牛, ATK×2 |
| | P3 | 20% HP | ATK×3 + 反伤20% |
| 老龟(81) | P2 | 75% HP | 翻转(全屏攻击) |
| | P3 | 50% HP | 水牢(限制移动) |

#### 狂暴计时器
```
enrage_timer: Boss战超时后触发
effect: 每秒ATK+5%, DEF+3%, 30秒后ATK×3
设计意图: DPS检测，防无限磨血
```

| 难度区间 | 狂暴触发 | 狂暴后存活窗口 |
|---------|---------|--------------|
| 1-9 | 120s | ~30s |
| 10-27 | 90s | ~20s |
| 28-54 | 60s | ~15s |
| 55-81 | 45s | ~10s |

### 3.4 Boss 战力门槛公式

```
recommended_power(n) = floor(boss_hp(n) / time_limit(n) × 1.2)
// 玩家需在限时内打出 120% Boss HP 的伤害（含容错）

gear_check(n):
  n <= 9:  凡品+5 以上
  n <= 18: 灵品+3 以上
  n <= 27: 灵品+8 / 仙品
  n <= 36: 仙品+5 以上
  n <= 45: 神品 以上
  n <= 54: 神品+8 / 混沌
  n <= 72: 混沌+5 以上
  n <= 81: 混沌+10 / 鸿蒙
```

## 6. 与现有系统的兼容性

| 现有系统 | 交互方式 |
|---------|---------|
| 装备系统(v2.0) | 锻造产出的装备进入同一背包系统，品质/强化体系不变 |
| 境界系统 | 采集点按境界解锁；境界倍率影响锻造武器的最终属性 |
| 取经进度 | 秘境按通关进度解锁；挂机产出按关卡范围分级 |
| 离线系统 | 离线期间采集点自动产出（效率×50%）；挂机材料照常计算 |
| 转世系统 | 转世保留锻造熟练度和图纸；材料和装备重置 |
| 角色系统 | 猪八戒采集加速、沙悟净采集量加成、悟空锻造暴击（双倍属性5%概率）|
