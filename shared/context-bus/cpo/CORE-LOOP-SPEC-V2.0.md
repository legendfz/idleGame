### 1.2 修为需求公式

**大境界 r（1~14）内第 s 小层级（1~9）的突破修为**：

```
xiuwei_required(r, s) = floor(base(r) × sub_scale(s))

base(r) = 100 × 10^(r-1) × 1.2^(r-1)
sub_scale(s) = 1.0 + (s-1) × 0.4 + (s-1)^2 × 0.05
```

**数值示例**：

| 境界 | 层级 | 修为需求 |
|------|------|---------|
| 凡人·一层 | (1,1) | 100 |
| 凡人·九层 | (1,9) | 556 |
| 练气·一层 | (2,1) | 1,200 |
| 练气·九层 | (2,9) | 6,672 |
| 筑基·一层 | (3,1) | 17,280 |
| 金丹·一层 | (4,1) | 298,598 |
| 元婴·一层 | (5,1) | 6,091,006 |
| 化神·一层 | (6,1) | 146,184,140 |
| 圣人·一层 | (14,1) | ~1.2 × 10^19 |
| 圣人·九层 | (14,9) | ~6.7 × 10^19 |

### 1.3 修炼速度

```
xiuwei_per_sec = base_speed × realm_multiplier × equipment_bonus × team_bonus × buff_bonus

base_speed = 1
realm_multiplier = 境界倍率（见 1.1 表）
equipment_bonus = 1 + Σ(装备修炼加成%)
team_bonus = 1 + Σ(队友被动修炼加成%)    // 唐僧: +15%
buff_bonus = 1 + Σ(临时增益%)             // 仙丹、神仙助力
```

**关键节点修炼速度**：

| 境界 | 每秒修为 | 到下一大境界需要时间 |
|------|---------|-------------------|
| 凡人·一 | 1 | ~10 分钟（到练气） |
| 练气·一 | 1.5 | ~3.2 小时 |
| 筑基·一 | 2.5 | ~33 小时 |
| 金丹·一 | 4 | ~17 天 |
| 元婴·一 | 7 | ~240 天（需装备+队友） |

> 设计意图：纯挂机到元婴需数月，但装备/队友/丹药可大幅缩短。金丹是 F2P 第一周目标。

### 1.4 离线收益计算

```
offline_xiuwei = offline_seconds × xiuwei_per_sec × offline_efficiency

offline_efficiency = 0.5（基础）
                   + 0.2（悟空"定身法"解锁后）
                   + equipment_offline_bonus
                   // 最高 1.0（不超过在线效率）

offline_seconds = min(actual_offline, 86400)  // 24h 上限

// 回归奖励（离线 > 8h）
bonus_xiuwei = offline_xiuwei × 0.1
bonus_gold = offline_gold × 0.1
```

**离线金币**：
```
offline_gold = offline_seconds × gold_per_sec × offline_efficiency
gold_per_sec = 扫荡关卡产出 / 90  // 假设每关 90 秒
```

## 2. 战斗系统详细设计

### 2.1 角色属性公式

```
attack(level, realm) = floor((10 + level × 3) × realm_multiplier × (1 + equip_atk%))
defense(level, realm) = floor((5 + level × 1.5) × realm_multiplier × (1 + equip_def%))
hp(level, realm) = floor((100 + level × 20) × realm_multiplier × (1 + equip_hp%))
```

`level` = 角色等级（1~999），由经验驱动
`realm_multiplier` = 当前大境界倍率

**Lv.1 各境界属性示例**：

| 境界 | 攻击 | 防御 | HP |
|------|------|------|-----|
| 凡人 | 13 | 6 | 120 |
| 练气 | 19 | 9 | 180 |
| 金丹 | 52 | 26 | 480 |
| 元婴 | 91 | 45 | 840 |
| 天仙 | 780 | 390 | 7,200 |

### 2.2 点击伤害

```
click_damage = base_click × (1 + click_bonus%)

base_click = attack × 0.5   // 基础 = 攻击力 50%

// 暴击（5% 基础概率）
crit_roll = random() < crit_rate
crit_damage = click_damage × crit_multiplier  // 默认 ×2.0

// 连击 buff
if consecutive_clicks >= 10 in 3 seconds:
    click_damage × 2.0 for 5 seconds
    show "连击加速！" effect
```

**悟空专属**：暴击率 +10%（总 15%），暴击伤害 ×2.5

### 2.3 自动攻击 DPS

```
auto_dps = attack × auto_speed × (1 + auto_bonus%)

auto_speed = 1.0 attacks/sec（基础）
           + speed_bonus（装备/技能）
           // 上限 5.0 attacks/sec

// 暴击融入 DPS
effective_dps = auto_dps × (1 + crit_rate × (crit_multiplier - 1))
```

### 2.4 Boss 血量曲线（81 难）

```
boss_hp(n) = floor(500 × 2.2^(n-1) × tier_bonus(n))

tier_bonus(n):
  n % 9 == 0 (章节Boss): × 3.0
  n % 3 == 0 (小节点):   × 1.5
  else:                   × 1.0
```

**关键节点 Boss HP**：

| 难数 | 描述 | Boss HP | 推荐战力 |
|------|------|---------|---------|
| 1 | 第 1 难 | 500 | 50 |
| 3 | 第一个小节点 | 3,630 | 350 |
| 9 | 第一章 Boss（寅将军） | 149,361 | 14,000 |
| 18 | 第二章 Boss（黄风大王） | 134M | 12.5M |
| 27 | 第三章 Boss（白骨精） | 120B | 11.2B |
| 36 | 第四章 Boss（红孩儿） | 107T | 10T |
| 45 | 第五章 Boss（牛魔王） | 9.6e16 | 9e15 |
| 54 | 第六章 Boss（大鹏） | 8.6e19 | 8e18 |
| 63 | 第七章 Boss（六耳猕猴） | 7.7e22 | 7.2e21 |
| 72 | 第八章 Boss（九头虫） | 6.9e25 | 6.4e24 |
| 81 | 最终 Boss（老龟翻转） | 6.2e28 | 5.8e27 |

> 指数增长设计核心：每 9 难 HP 增长约 1000 倍，对应一个完整大境界突破。

### 2.5 Boss 战时间限制

| 难度区间 | 时限 | 设计意图 |
|---------|------|---------|
| 1-9 | 120 秒 | 新手友好 |
| 10-27 | 90 秒 | 中等紧凑 |
| 28-54 | 60 秒 | 需要策略 |
| 55-81 | 45 秒 | 高压 DPS check |

### 2.6 战斗节奏设计

每一难 = **3~5 波小怪 + 1 Boss**，总节奏：

| 阶段 | 时间 | 玩家动作 |
|------|------|---------|
| 小怪 1-3 波 | 15-30 秒 | 自动攻击 + 偶尔点击 |
| 精英怪 | 10-20 秒 | 需要关注 + 主动点击 |
| Boss 登场 | 3 秒 | 预警动画 |
| Boss 战 | 45-120 秒 | 密集点击 + 技能释放 + 闪避 |
| 结算 | 5 秒 | 奖励展示 |

**每关总时长目标**：60~180 秒（碾压~刚好达标）

## 3. 收集系统规格

### 3.1 装备掉落概率

**小怪掉落率**：5%（每波判定一次）
**Boss 掉落率**：100%（必掉 1 件）+ 30% 额外掉落

| 品质 | 小怪掉落 | Boss 掉落 | Boss 首通 | 章节 Boss |
|------|---------|----------|----------|----------|
| 凡品 | 70% | 35% | 15% | 5% |
| 灵品 | 24% | 35% | 30% | 20% |
| 仙品 | 5% | 20% | 30% | 35% |
| 神品 | 0.9% | 8% | 18% | 25% |
| 混沌 | 0.1% | 1.8% | 6% | 12% |
| 鸿蒙 | 0% | 0.2% | 1% | 3% |

**装备等级** = 当前难度 ± rand(2)

### 3.2 法宝获取

| 法宝 | 途径 | 概率/条件 |
|------|------|----------|
| 紧箍咒 | 第 1 难剧情 | 100%（固定） |
| 金箍棒 | 第 3 难首通 | 100%（固定） |
| 芭蕉扇 | 第 36 难首通 | 100%（固定） |
| 紫金红葫芦 | 第 20 难 Boss 掉落 | 15% |
| 照妖镜 | 第 13 难 Boss 掉落 | 10% |
| 定海神针 | 隐藏成就 | 条件触发 |

### 3.3 妖怪招降

```
capture_chance(n) = base_rate × (1 + charisma_bonus) × weakness_multiplier

base_rate = 0.05  // 5% 基础
charisma_bonus = 唐僧被动 0.15 + 装备加成
weakness_multiplier:
  满足特殊条件: × 3.0（如红孩儿需观音法水）
  击败 3 次以上: × 1.5
  else: × 1.0
```

每次击败 Boss 自动判定招降。招降后进入队友栏，提供被动加成。

### 3.4 丹药/材料产出

| 材料 | 来源 | 每小时期望 | 用途 |
|------|------|-----------|------|
| 聚灵丹 | 1-9 难掉落（20%/Boss） | ~3 个 | 练气突破 |
| 筑基丹 | 10-18 难掉落（15%） | ~2 个 | 筑基突破 |
| 金丹草 | 19-27 难掉落（10%） | ~1 个 | 金丹突破 |
| 天材地宝 | 章节 Boss 首通 | 首通固定 | 高级突破 |
| 炼丹材料 | 所有关卡（30%） | ~10 个 | 炼丹系统 |

## 4. 转世（Prestige）系统

### 4.1 触发条件
- 通关第 81 难
- 境界达到「圣人」

### 4.2 转世收益公式

```
foyuan(佛缘) = floor(total_xiuwei_earned^0.4 × prestige_count_bonus)

prestige_count_bonus = 1.0 + prestige_count × 0.1
// 第 1 次转世: ×1.1, 第 10 次: ×2.0
```

**首次转世佛缘预估**：
- 通关 81 难时累计修为 ≈ 10^20
- 佛缘 = floor((10^20)^0.4 × 1.1) ≈ 11,000

## 5. 交互流程图

### 5.1 新玩家前 5 分钟

```
0:00  ┌─ 开场文案「大唐贞观年间…」(可跳过)
0:15  ├─ 界面出现：修为 0, 境界「凡人·一层」
      │  提示：「点击屏幕开始修炼」
0:20  ├─ 玩家首次点击 → 修为 +10（点击产出）
      │  提示：「继续点击积累修为」
0:40  ├─ 修为达到 50 → 自动弹出「新手引导：自动修炼」
      │  开启每秒 +1 修为
1:00  ├─ 修为达到 100 → 提示「可突破！」按钮高亮
      │  玩家点击突破 → 动画 → 「练气·一层」
      │  解锁自动攻击
1:30  ├─ 提示「第 1 难：长安城外」→ 自动进入战斗
      │  3 波小怪 + Boss（寅将军，HP 500）
3:00  ├─ 通关第 1 难 → 掉落装备（必掉凡品武器）
      │  提示穿戴装备 → 属性提升
3:30  ├─ 提示「紧箍咒获得！」→ 法宝系统解锁
4:00  ├─ 自动进入第 2 难 → 战斗
5:00  └─ 第 2 难通关 → 引导结束，自由探索
```

### 5.2 日常游戏循环

```
┌─ 打开游戏
│
├─ [离线收益弹窗]
│  显示：离线时长/修为/金币/可能的装备
│  [领取] → 资源入账
│
├─ [主界面]
│  查看：修为产出速度、当前境界、取经进度
│  │
│  ├─ 修为满 → [突破境界] → 动画 → 速度提升
│  │
│  ├─ [进入关卡] → 战斗（自动+点击）
│  │  └─ 通关 → 掉落奖励 → 下一关 or 返回
│  │
│  ├─ [背包] → 穿戴/强化/分解装备
│  │
│  ├─ [角色] → 查看属性/切换角色/技能
│  │
│  └─ [扫荡] (3星关卡) → 快速获取资源
│
├─ [每日任务] → 完成获取灵石
│
└─ 退出（自动存档）→ 离线修炼继续
```

### 5.3 关键决策点

| 时机 | 决策 | 选项 |
|------|------|------|
| 修为满 | 突破 or 继续积累 | 突破（消耗修为+材料）/ 等攒更多修为 |
| Boss 战 | 主动打 or 刷小怪 | 挑战 Boss / 扫荡前面关卡攒资源 |
| 装备掉落 | 穿戴 or 分解 or 强化旧装 | 取决于属性对比 |
| 佛缘消费 | 哪项升级优先 | 修炼加速 vs 攻击 vs 起始境界 |
| 角色切换 | 谁当主控 | 悟空(DPS) / 唐僧(修炼) / 八戒(金币) |
