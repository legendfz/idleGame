# PRD v10.0 —「归元」综合质量版

**版本**：10.0 | **作者**：CPO | **日期**：2026-03-02
**状态**：Draft → 待 CEO 审批
**依赖**：PRD-V2.0 ~ V9.0 全系统

---

## 1. 版本定位

v10.0 不新增大型系统，聚焦三大优化方向：

```
系统联动 → 打通已有系统的协同效果，提升深度
数值平衡 → 全面审查产出/消耗，修正失衡点
新手引导 → 降低上手门槛，提高留存
```

## 2. 系统联动设计

### 2.1 联动矩阵

| 来源系统 | → 目标系统 | 联动效果 | 实现方式 |
|---------|-----------|---------|---------|
| 天赋树(机缘) | → 锻造 | 机缘路线「鸿运当头」锻造成功率+8% | forge.ts 读取天赋加成 |
| 天赋树(力道) | → 通天塔 | 力道「战神意志」Boss伤害+20% | tower 战斗引用天赋 |
| 天赋树(心法) | → 离线 | 心法「引气入体」离线效率+15% | offlineCalc 读取天赋 |
| 神通 | → 灵兽战力 | 每解锁1个神通，携带灵兽加成+2% | petBonus 计算时叠加 |
| 成就 | → 商店 | 达成指定成就解锁专属商品 | shopStore 检查成就 |
| 通天塔 | → 每日任务 | 通天塔层数每 50 层，每日任务基础奖励+10% | dailyQuest 读取 towerStore |
| 转世次数 | → 灵兽等级上限 | 基础上限10，每次转世+1（上限15） | petStore 读取转世次数 |
| 灵兽 | → 天赋效果 | 携带麒麟时，机缘路线效果额外+10% | talentCalc 读取 activePet |
| 里程碑 | → 神通 | 达成战斗里程碑「金」，悟道值+100 | milestone 完成时触发 |

### 2.2 成就解锁商品

| 成就 | 解锁商品 | 商铺位置 | 价格 |
|------|---------|---------|------|
| 「塔中豪杰」(通天塔100层) | 🐲 灵兽精华 | 灵石铺 | 30 灵石 |

**灵兽精华**：灵兽经验×2（1小时）
**万法归宗卷**：天赋重置冷却清零

**悟道仙丹**：悟道值+200

### 2.3 联动公式汇总

```typescript
// 锻造成功率
forgeRate = baseRate × (1 + talentForgeBonus) × (1 + petBreakthroughPill)

// 灵兽加成
petBonus = basePetBonus × (1 + unlockedSkillCount × 0.02) × (1 + isQilin ? talentFortuneExtra : 0)

// 每日任务奖励
dailyReward = baseReward × (1 + floor(towerHighest / 50) × 0.10)

// 灵兽等级上限
petMaxLevel = min(15, 10 + reincarnationCount)

// 战力（更新）
power = (atk × 2 + def × 1.5 + hp × 0.1) × realmMul × petBonus × (1 + talentPowerBonus)
```

## 3. 数值平衡审查

### 3.1 资源流转分析

#### 金币经济

| 阶段 | 主要产出 | 产出/小时 | 主要消耗 | 消耗/小时 | 盈余 |
|------|---------|----------|---------|----------|------|
| 前期(练气~筑基) | 战斗掉落+离线 | ~5,000 | 装备强化 | ~3,000 | +2,000 ✅ |
| 中期(金丹~化神) | 战斗+通天塔扫荡 | ~30,000 | 强化+商铺 | ~25,000 | +5,000 ✅ |
| 后期(大乘~圣人) | 通天塔+排行榜 | ~150,000 | 高级锻造+商铺 | ~180,000 | -30,000 ⚠️ |

**问题**：后期金币消耗大于产出，需调整。
**建议**：通天塔 100+ 层金币奖励从 12,000×层 提升至 15,000×层。

#### 灵石经济

| 阶段 | 主要产出/日 | 主要消耗/日 | 盈余 |
|------|-----------|-----------|------|
| 前期 | 20~30 (任务+战斗) | 0~20 (商铺偶买) | +10~30 ✅ |
| 中期 | 60~100 (塔+排行+任务) | 50~80 (购买次数+商铺) | +10~20 ✅ |
| 后期 | 150~250 (三榜+高层塔) | 100~200 (高级商品+天赋重置) | +0~50 ⚠️ |

**问题**：后期灵石较紧张但可控，逼迫玩家做选择，属于健康设计。
**建议**：保持现状。

#### 灵兽丹经济

| 来源 | 数量/日 |
|------|--------|
| 排行榜第1(每榜) | 1×3 = 3 |

### 4.1 引导架构

5 步引导，每步有明确目标、交互提示和完成奖励。引导以对话气泡形式出现（符合纯文字竖屏风格）。

### 4.3 引导状态管理

```typescript
interface TutorialState {
  currentStep: number;    // 0=未开始, 1-5=进行中, 6=已完成
  stepsCompleted: boolean[];  // [false,false,false,false,false]
  rewards Claimed: boolean[]; // 各步奖励是否已领
}
```

- 存入 `getFullState()` / `loadFullState()`
- 引导完成后不再弹出
- 老存档（已有进度）直接标记 `currentStep=6`（跳过引导）

### 4.4 引导 UI 规则

- 引导气泡覆盖在当前界面上方
- 有「跳过引导」按钮（随时可跳过全部）
- 跳过时不发放未领奖励
- 气泡文案使用西游记师父语气
