## 1. 六道轮回面板

### 设计令牌
```css
--dao-heaven: #FFD700;  /* 天道·金 */
--dao-human:  #42A5F5;  /* 人道·蓝 */
--dao-asura:  #EF5350;  /* 修罗道·红 */
--dao-animal: #66BB6A;  /* 畜生道·绿 */
--dao-ghost:  #AB47BC;  /* 饿鬼道·紫 */
--dao-hell:   #B71C1C;  /* 地狱道·暗红 */
```

### 布局
```
          ☀️ 天道
     👹            👤
    地狱道        人道
         ☸️ 中心
     👻            ⚔️
    饿鬼道        修罗道
          🐂 畜生道
```
- 圆形排列 280×280px，中心 ☸️ 60px 圆形
- 6 节点各 56px 圆形，nth-child 定位于 12/2/4/6/8/10 点钟方向
- 三态：locked(灰25%) → available(道色边框) → selected(放大+光晕) → completed(半透明+✓角标)

### 交互
- 点击节点 → 底部弹出确认弹窗（含奖励预览+警告文字）
- 确认轮回 → `reincarnate` 动画（闪白→黑屏→淡入，1.5s）
- 道果列表：已解锁金色渐变背景，未解锁灰化40%

### CSS 文件
`src/styles/reincarnation.css`（7.9KB）

## 2. 秘境深层 UI

### 布局
```
┌─────────────────────┐
│ 🌀 无尽秘境          │
│ 最高通关: 第42层     │
│ [████████░░░░] 42%   │ ← 总进度 /100
│                      │
│ ┌─ 选择层数 ────────┐│
│ │  [−]   47   [+]   ││ ← Boss层金色数字
│ │    ⚔️ Boss层       ││
│ └────────────────────┘│
│                      │
│ [怪物增伤] [减速] [双倍金币] │ ← 词缀标签(红/绿/金)
│                      │
│ [  🌀 挑战  ] [扫荡] │
│                      │
│ ┌─ 奖励预览 ────────┐│
│ │ 灵石    +5,200    ││
│ │ ★ Boss宝箱 随机装备││ ← Boss奖励金色高亮
│ └────────────────────┘│
└─────────────────────┘
```

### 核心组件
- **层数选择器**：±按钮（36px圆形）+ 中间大数字（28px），Boss层(每10层)金色+text-shadow
- **词缀标签**：pill 胶囊，buff绿/debuff红/中性金
- **进度条**：紫色渐变(#7C4DFF→#B388FF)，10px高
- **挑战按钮**：紫色渐变，2:1占比；扫荡按钮：紫色描边

### CSS 文件
`src/styles/abyss.css`（5.3KB）

## 3. 排行榜面板

### 布局
```
┌─────────────────────┐
│ 🏆 排行榜            │
│ [境界] [秘境] [战力]  │ ← 三Tab分段控件
│                      │
│     🥈    🥇    🥉    │ ← 三甲 podium
│    镇元子  如来  观音  │
│    筑基   大乘  菩萨  │
│                      │
│ 4  🐒 孙悟空  练气九  │
│ 5  🐷 猪八戒  练气七  │
│ 6  🎭 红孩儿  练气五  │ ← NPC西游记风格
│ 7  ⭐ 你      练气三  │ ← 自己金色高亮
│ ...                  │
│ ┌─ 我的排名: #7 ─────┐│ ← sticky底栏
└─────────────────────┘
```

### 核心组件
- **三甲 Podium**：#1 大头像60px+金边光晕，#2/#3 48px+银/铜边
- **排名行**：flex布局（排名28px + 头像32px + 名字flex:1 + 数值），自己行金色底色+金色边框
- **自己底栏**：sticky固定，显示排名和数值
