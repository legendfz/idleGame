## 1. 天赋树面板

**复用 v5.0 `talent.css`**，扩展路线配色：

| 路线 | 图标 | Tab色 | 节点色 |
|------|------|-------|--------|
| 力道 | ⚔️ | `--color-primary`(红) | 红色光晕 |
| 心法 | 🧘 | `--color-accent`(金) | 金色光晕 |
| 机缘 | 🎲 | `--color-dungeon`(紫) | 紫色光晕 |

已有样式覆盖：`.talent-branch-tab.battle/cultivate/craft`，新增第三色系在组件层用 `craft` class 映射为紫色即可（或追加 `.talent-branch-tab.fortune.active { background: var(--color-dungeon); }`）。

连线发光效果：`.talent-connector.active` 已支持金色发光。

## 2. 神通技能面板

### 布局
```
┌─────────────────────────────┐
│ ✨ 神通      悟道值 ████ 72%│
├─────────────────────────────┤
│ ┌────┐ ┌────┐ ┌────┐       │
│ │ 🔥 │ │ ❄️ │ │ ⚡ │       │ ← 3×2宫格
│ │火眼 │ │冰心 │ │雷法 │       │
│ │⭐⭐⭐│ │⭐⭐☆│ │⭐☆☆│       │
│ └────┘ └────┘ └────┘       │
│ ┌────┐ ┌────┐ ┌────┐       │
│ │ 🌪️ │ │ 🛡️ │ │ 🔒 │       │
│ │风行 │ │金身 │ │???  │       │
│ │⭐☆☆│ │ CD │ │锁定 │       │
│ └────┘ └────┘ └────┘       │
└─────────────────────────────┘
```

### 核心组件
- **6宫格**：3列grid（≤374px降2列）
- **技能卡三态**：locked(灰30%) → normal(卡片) → active(金色边框+skillGlow脉冲2.5s)
- **冷却环**：SVG circle覆盖在icon上，`stroke-dashoffset` 控制进度，红色冷却秒数居中
- **等级星星**：实心⭐(accent色) + 空心☆(border色)
- **悟道值**：右上角mini进度条(100px宽，紫色渐变)
- **详情弹窗**：底部sheet，效果列表+升级按钮

### CSS: `src/styles/skill.css`（5.9KB）

## 3. 策略选择面板

### 布局
```
┌─────────────────────────────┐
│ 🎯 战斗策略                  │
│ 选择你的战斗风格              │
├─────────────────────────────┤
│ ┌──────┐┌──────┐┌──────┐   │
│ │  ⚔️  ││  ⚖️  ││  🛡️  │   │ ← 3卡片横排
│ │ 狂攻 ││ 均衡 ││ 防御 │   │
│ │攻+30%││ 默认 ││防+30%│   │
│ │[激活]││      ││      │   │
│ └──────┘└──────┘└──────┘   │
│                              │
│ ┌─ 当前效果 ───────────────┐│
│ │ 攻击力   +30%   ↑        ││ ← buff绿
│ │ 防御力   -15%   ↓        ││ ← nerf红
│ │ 暴击率   +10%   ↑        ││
│ └──────────────────────────┘│
└─────────────────────────────┘
```

### 核心组件
- **3卡片横排**：`flex: gap: sm`，三策略各自配色（aggressive红/balanced金/defensive蓝）
- **激活态**：品牌色边框+底部渐变+12px光晕
- **icon圆形**：52px，激活时品牌色边框+背景
- **效果详情**：buff值绿色/nerf值红色/neutral白色
- **切换动画**：`strategySwitch` scale弹跳 0.4s

### CSS: `src/styles/strategy.css`（4.3KB）
