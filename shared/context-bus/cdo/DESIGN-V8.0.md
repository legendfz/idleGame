### 核心组件
- **塔形进度**：`column-reverse` flex，每层 8px 色段（cleared 金/current 红脉冲/locked 灰），Boss层 12px+金边
- **当前层**：大数字(28px)，Boss层金色+text-shadow，敌人预览(icon+名称+属性)
- **剩余次数**：pill 胶囊，0 次时红色
- **挑战按钮**：红色渐变(primary→深红)，2:1 占比；扫荡按钮红色描边

## 3. 灵兽面板

### 设计令牌
```css
--pet-azure:  #00BCD4;  /* 青龙 */
--pet-white:  #E0E0E0;  /* 白虎 */
--pet-vermil: #F44336;  /* 朱雀 */
--pet-black:  #607D8B;  /* 玄武 */
--pet-golden: #FFD700;  /* 麒麟 */
```

### 布局
```
┌─────────────────────────────┐
│ 🐉 灵兽      💊 灵兽丹 ×12  │
├─────────────────────────────┤
│ ┌─ 当前携带 ───────────────┐│
│ │ 🐉 青龙 Lv.5             ││
│ │ [攻+8%] [暴击+3%]        ││ ← 青色pill标签
│ └──────────────────────────┘│
│                              │
│ ┌─🐉── 青龙 ── Lv.5 ──────┐│
│ │ 攻击+8%  暴击率+3%       ││ ← 青色左边框+图标底色
│ │ [████████░░] 80/100      ││ ← 青色经验条
│ │         [携带中] [升级💊3]││
│ └──────────────────────────┘│
│ ┌─🐅── 白虎 ── Lv.3 ──────┐│
│ │ 防御+5%  血量+4%         ││ ← 白色系
│ │ [██████░░░░] 45/80       ││
│ │           [携带] [升级💊2]││
│ └──────────────────────────┘│
│ ┌─🔒── 朱雀 ── 未解锁 ────┐│
│ │ 通关通天塔30层解锁        ││ ← 灰化35%
│ └──────────────────────────┘│
│ ...                          │
└─────────────────────────────┘
```

### 核心组件
- **五兽卡片**：4px 左边框用各自品牌色（azure/white/vermil/black/golden）
- **三态**：locked(灰+灰度icon) → owned(品牌色左边框) → equipped(品牌色光晕+渐变底色)
- **属性标签**：pill 胶囊，各兽品牌色文字+透明底色
- **经验条**：4px，各兽品牌色填充
- **操作按钮**：携带(金色) / 携带中(绿色) + 升级(金色描边，disabled灰)
- **灵兽丹**：顶部 pill 胶囊

### CSS: `src/styles/pet.css`（7.3KB）
