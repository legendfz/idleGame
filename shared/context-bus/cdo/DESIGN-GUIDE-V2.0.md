## 1. 视觉风格建议

### ⭐ 推荐方案：Q版国风

**理由：**
- Q 版角色降低美术成本，适合小团队
- 国风配色和纹样保留西游记文化氛围
- 与 Idle Game 轻松玩法调性匹配
- 像素风虽成本更低但难以表达西游记的华丽感
- 写实国风成本过高，不适合 MVP

### 风格关键词
- **整体**：Q版 + 中国风 + 暖色调
- **线条**：圆润可爱，2-3px 描边
- **配色**：朱红、金黄、翠绿、靛蓝为主色调
- **纹样**：祥云、莲花、回纹作为装饰元素
- **氛围**：轻松幽默，不过于严肃

### 参考游戏
- 《梦幻西游》Q 版风格
- 《阴阳师》UI 质感
- 《最强蜗牛》Idle 类 UI 布局
- Cookie Clicker / Idle Miner 的点击反馈

## 2. 核心角色概念设计方向

### 唐僧
- 圆脸慈眉，大眼睛
- 金色袈裟，手持禅杖
- 头顶光环（佛光），随境界变强
- 表情：温和坚定

### 孙悟空
- 尖脸，桃心眼，嘴角上翘
- 虎皮裙 + 锁子甲，手持金箍棒
- 火眼金睛发光效果
- 表情：调皮自信，战斗时凶猛

### 猪八戒
- 圆胖身体，大耳朵，猪鼻
- 黑色僧衣，九齿钉耙
- 经常有食物元素出现
- 表情：憨厚贪吃

### 沙悟净
- 方脸，胡须，蓝色皮肤
- 念珠 + 降妖宝杖
- 背着行李（视觉彩蛋：背包 = 玩家背包）
- 表情：沉稳可靠

### 白龙马
- 小白龙马形态，Q版可爱
- 鬃毛有水波纹效果
- 可变回龙形态（特殊技能时）

### 妖怪设计方向
- 每个妖怪保留原著特征但 Q 版化
- Boss 体型 = 普通妖怪 2-3 倍
- 被击败时有趣的倒地动画

## 3. UI 风格指南初稿

### 3.1 色彩系统

```
主色：
- 朱红 #C13B3B — 主按钮、重要操作
- 金黄 #D4A843 — 货币、品质高光
- 靛蓝 #2B4C7E — 背景、面板

辅色：
- 翠绿 #3A7D44 — 成功、生命值
- 紫色 #6B3FA0 — 稀有品质
- 橙色 #E67E22 — 警告、限时

中性色：
- 深棕 #3E2723 — 文字
- 米白 #FFF8E7 — 面板底色
- 浅金 #F5E6C8 — 卡片背景
```

### 3.2 品质颜色（沿用 v1）

| 品质 | 颜色 | 色值 |
|------|------|------|
| 凡品 | 灰色 | #9E9E9E |
| 灵品 | 绿色 | #4CAF50 |
| 仙品 | 蓝色 | #2196F3 |
| 神品 | 紫色 | #9C27B0 |
| 混沌 | 橙色 | #FF9800 |
| 鸿蒙 | 金色 | #FFD700 |

### 3.3 字体

- 标题：思源宋体（Noto Serif SC）— 国风感
- 正文：思源黑体（Noto Sans SC）— 可读性
- 数字：Roboto Mono — 等宽，数值清晰
- 降级：系统默认中文字体

### 3.4 UI 布局（主界面）

```
┌─────────────────────────────┐
│  [角色名] Lv.15  练气九层   │ ← 顶栏：角色信息

├─────────────────────────────┤
│                             │

│     [ 点击区域/战斗 ]        │
│                             │
│  ═══●═══════════════════    │ ← 取经进度条
│  第3难 / 81                  │
├─────────────────────────────┤

```

### 3.5 动画与反馈

- **点击反馈**：涟漪 + 伤害数字弹出
- **暴击**：金色大字 + 屏幕微震
- **突破**：全屏金光 + 境界文字飞入
- **掉落**：装备图标从天而降 + 品质光柱
- **按钮**：按下缩小 95%，弹起恢复

### 3.6 组件规范

| 组件 | 圆角 | 阴影 | 边框 |
|------|------|------|------|
| 卡片 | 12px | 0 2px 8px rgba(0,0,0,0.1) | 1px #E0D5C0 |
| 按钮 | 8px | 0 2px 4px rgba(0,0,0,0.2) | none |
| 面板 | 16px | 0 4px 16px rgba(0,0,0,0.15) | 2px #D4A843 |
| 输入框 | 6px | inset | 1px #C0B090 |

### 3.7 图标风格
- 线性图标 + 国风装饰
- 2px 描边，圆角端点
- 尺寸：24×24 (nav), 32×32 (action), 48×48 (feature)
