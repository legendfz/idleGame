# CDO: UI 设计方案摘要

**日期**：2026-03-01
**状态**：v1.0 完成，待反馈

## 核心决策
1. **符号体系**：Emoji + Unicode 符号表示角色/装备/属性，品质用 ○●◆★✦✧ 前缀 + 颜色区分
2. **布局**：固定顶栏（场景+资源）+ 可滚动内容区 + 固定底部 5-Tab 导航
3. **5 Tab**：🗡️战斗 | 🐒队伍 | 🗺️旅途 | 📦背包 | ⚙️更多
4. **色彩**：暗色主题（深靛蓝底 #1a1a2e，金色强调 #f0c040）为默认，亮色主题（宣纸黄 #f5f0e8）可选
5. **战斗界面**：上方敌人血条 → 中间战斗日志自动滚动 → 下方挂机收益
6. **文字动画**：用框线/分隔符/特殊排版区分普攻/暴击/技能/Boss/升级等反馈
