# v2.2 开发计划 — UI 改进迭代

## 目标
实现 CDO 的 5 项 P2 UI 改进，提升视觉反馈和设计一致性

## 开发顺序（按 CDO 建议）

| 任务 | 来源 | 估时 | 说明 |

### Sprint 3：地图重构（~4h）

|------|------|------|------|
| 章节折叠分组 | P2-04 | 2h | 81 难按章节 accordion |

| 章节进度 mini bar | P2-04 | 0.5h | 标题右侧进度展示 |

## 总估时：~11.5h（3 个 Sprint）

## 技术要点
- 纯 CSS 动画，不引入动画库
- 所有样式使用设计指南变量（--color-*, --sp-*, --fs-*）
- 保持组件结构不变，仅增强视觉层

## 验收标准
- 所有硬编码值替换为主题变量
- 突破动画完整 800ms 序列
- 导航栏胶囊滑动流畅
- 地图可折叠，当前章节默认展开
- 无 TypeScript 编译错误，构建通过
