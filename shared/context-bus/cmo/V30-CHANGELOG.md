## 🔨 武器锻造系统

从今天起，你的武器由你亲手打造。

锻造台开炉。收集材料，选择配方，一锤定乾坤。

### 8 大锻造配方

| 配方 | 材料 | 保底品质 | 成功率 |
|------|------|---------|--------|
| ⛏️ 铁剑锻造 | 铁矿 ×5 | ⬜ 凡品 | 95% |
| 💎 灵刃锻造 | 铁矿 ×10 + 灵石碎片 ×3 | 🟢 灵品 | 85% |
| 🪵 灵纹轻甲 | 灵木 ×8 + 铁矿 ×5 | 🟢 灵品 | 80% |
| ⭐ 星铁长枪 | 星铁 ×8 + 灵木 ×5 | 🔵 仙品 | 70% |
| 🐉 龙鳞铠甲 | 龙鳞 ×12 + 星铁 ×5 | 🟣 神品 | 55% |
| 🪶 凤羽冠 | 凤羽 ×6 + 天材 ×4 | 🟣 神品 | 50% |
| 🌀 混沌指环 | 混沌精华 ×5 + 佛光石 ×8 | 🟡 混沌 | 35% |
| 🔥 鸿蒙法杖 | 鸿蒙之核 ×3 + 混沌精华 ×10 + 圣物 ×5 | 🔴 鸿蒙 | 15% |

- 锻造失败返还部分材料
- 成功可触发随机附加属性（暴击/吸血/穿透等）
- 锻造等级越高，解锁更多配方

## ⛏️ 采集系统

四大西游主题采集节点，不同地点产出不同材料：

| 节点 | 材料 | 品级 |
|------|------|------|
| 🏔️ 五行山 | ⛏️ 铁矿 · 💎 灵石碎片 | ★☆☆☆☆ |
| 🌺 花果山 | 🪵 灵木 · 🍑 仙果 | ★★☆☆☆ |
| ☁️ 天宫 | ⭐ 星铁 · ✨ 天材 | ★★★☆☆ |
| 🏛️ 灵山 | 🔮 佛光石 · 📿 圣物 | ★★★★☆ |

- 采集有冷却时间，定时收取
- 挂机自动产出（最多 24h）
- 高级节点需对应境界解锁

## 🐉 Boss 机制升级

后期 Boss 新增 7 种战斗机制，不再只是血条长：

| 机制 | 效果 |
|------|------|
| 🛡️ 免疫 | 特定伤害类型无效，需换武器 |
| ⚡ 反击 | 受击时反伤一定比例 |
| 🔄 多阶段 | 血量阈值触发形态变化 |
| 💀 狂暴 | 超时后攻击力暴涨 |
| 🌀 护盾 | 阶段性无敌，需等待破盾窗口 |
| 🎯 弱点 | 特定属性武器造成额外伤害 |
| ☠️ 秒杀技 | 技能预警+闪避，吃到即死 |

你需要策略、需要对的装备、需要亲手锻造的神兵。

## 🔥 炼化系统

低级材料合成高级材料：

```
低级材料 ×10  → 炼化 → 高品级材料 ×1
🦴 妖骨 ×10   → ⚗️ → 🟢 玉髓 ×1
```

Boss 掉落的 🦴 妖骨也能炼化为有用材料。
