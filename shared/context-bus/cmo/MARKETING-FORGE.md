## 4. 新手攻略 — 锻造系统入门指南

**标题**：「从矿石到神兵 — 锻造系统完全指南」

### 一、锻造系统是什么？

v3.0 新增的核心玩法。你不再只是捡别人掉的装备——你亲手收集材料、选择配方、锻造出专属武器。不同材料组合产出不同属性的武器。

### 二、快速上手（3 步走）

```
Step 1️⃣ 收集材料
  → 采集（矿脉/药田/灵泉）
  → 打怪掉落
  → 每日任务奖励
  → 挂机自动产出

Step 2️⃣ 选择配方
  → 打开锻造台
  → 查看可用配方（材料足够的会高亮）
  → 不同配方 = 不同武器属性

Step 3️⃣ 开始锻造
  → 点击锻造
  → 等待结果
  → 成功 → 🎉 新武器入手
  → 失败 → 返还部分材料，再来一次
```

### 三、材料获取优先级

| 优先级 | 途径 | 说明 | 建议 |
|--------|------|------|------|
| ⭐⭐⭐ | 每日任务 | 稳定产出，每天必做 | 每天做完 |
| ⭐⭐⭐ | 挂机产出 | 睡觉就能攒 | 选对挂机地点 |
| ⭐⭐ | 采集 | 有 CD，定时收 | 通勤/午休时操作 |
| ⭐⭐ | Boss 掉落 | 稀有材料主要来源 | 能打就打 |
| ⭐ | 秘境探索 | 限时+限次，难度高 | 战力够再进 |
| ⭐ | 炼化合成 | 低级→高级，转化率有损耗 | 低级材料多了再炼 |

### 四、锻造小技巧

1. **先凑齐材料再开炉** — 别一有材料就锻造，攒齐高品质配方更划算
2. **关注每日任务** — 稳定材料来源，别忘了做
3. **挂机地点要换** — 不同地点产出不同材料，按需要切换
4. **失败别气馁** — 返还材料 + 积累经验，高品质锻造本来就需要多次尝试
5. **打 Boss 前看机制** — 新 Boss 有免疫/反击，带错武器白打一场

### 五、常见问题

**Q: 锻造失败装备会消失吗？**
A: 不会。失败只消耗材料（部分返还），不影响已有装备。

**Q: 材料背包会占装备空间吗？**
A: 不会。v3.0 材料背包独立分页。

**Q: 挂机产出的材料有上限吗？**
A: 和离线收益一样，最多积累 24 小时。

**Q: 高品质武器一定要稀有材料吗？**
A: 是的。🟣 神品以上需要 Boss 掉落或秘境的稀有材料。
