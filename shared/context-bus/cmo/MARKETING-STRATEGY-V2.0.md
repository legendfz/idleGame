## 1. 市场定位

### 1.1 产品定位语

> **「九九八十一难，一指禅的修行」**
> — 用最少的操作，走最远的取经路

备选：
- 「挂机修仙，点击取经」
- 「你负责闭眼修炼，悟空负责降妖除魔」
- 「零图片，纯文字，九九八十一难的放置西游」

### 1.2 核心卖点（5 个）

| # | 卖点 | 一句话 |
|---|------|--------|
| 1 | **九九八十一难完整取经线** | 不是碎片化关卡，是完整的西游故事，从长安到灵山 |
| 2 | **纯文字零图片** | 极致轻量（~150KB），秒开，想象力就是最好的画面 |
| 3 | **挂机+点击双模式** | 挂机不无聊（有修炼进度），主动玩有操作感（Boss 限时战+闪避） |
| 4 | **深度养成** | 5 角色×6 装备槽×技能树×14 个修炼境界×转世系统 |
| 5 | **免费无广告** | 开源、透明、localStorage 存档，数据完全属于玩家 |

### 1.3 差异化优势

#### vs 其他 Idle Game

| 维度 | 主流 Idle Game | 西游 Idle v2.0 |
|------|---------------|---------------|
| 美术 | 重度像素/2D/3D | 零图片纯文字，极致差异化 |
| 叙事 | 弱/无/碎片化 | 81 难完整西游主线 |
| 角色 | 收集向，数百英雄 | 5 主角+可招降妖怪，少而精 |
| 体积 | 100MB-1GB | ~150KB |
| 商业化 | 广告/内购/抽卡 | 完全免费，开源 |
| 文化深度 | 通用奇幻 | 中国古典名著，每一难有原著呼应 |

#### vs 其他西游题材

| 维度 | 西游卡牌/MMO | 西游 Idle v2.0 |
|------|-------------|---------------|
| 同质化 | 严重换皮 | 独创纯文字形态 |
| 时间投入 | 需要肝 | 挂机为主，碎片时间 |
| 付费压力 | P2W 严重 | 零付费 |
| IP 使用 | 表面套皮 | 还原 81 难故事线+角色性格 |

**核心差异**：市场上没有「纯文字 × 完整西游 81 难 × Idle」的产品。我们开创品类。

## 2. 目标用户画像

### 2.1 主要用户群（70%）

**「通勤修仙党」**

| 维度 | 描述 |
|------|------|
| 年龄 | 20-32 岁 |
| 地域 | 中国一二线城市 |
| 职业 | 上班族/大学生 |
| 设备 | Android/iOS，中端手机 |
| 游戏偏好 | 放置/挂机/轻度 RPG，日均游戏 15-30 分钟 |
| 行为 | 通勤/午休/睡前打开看收益，偶尔手动打 Boss |
| 痛点 | 重度游戏没时间玩，轻度游戏没深度 |
| 需求 | 碎片时间有成长感，不肝但有追求 |

### 2.2 次要用户群（20%）

**「文字游戏鉴赏家」**

| 维度 | 描述 |
|------|------|
| 年龄 | 22-40 岁 |
| 地域 | 全球（中+英） |
| 特征 | 玩过 A Dark Room / Candy Box / Universal Paperclips |
| 活跃平台 | Reddit / Hacker News / V2EX / 少数派 |
| 动机 | 被「零图片」的独特形式吸引 |
| 痛点 | 主流游戏审美疲劳，寻找新鲜体验 |

### 2.3 边缘用户群（10%）

**「西游文化粉」**

| 维度 | 描述 |
|------|------|
| 年龄 | 18-45 岁 |
| 特征 | 西游记/悟空传/黑神话悟空 爱好者 |
| 动机 | IP 吸引力 + 81 难完整还原 |
| 活跃平台 | B 站 / 小红书 / 微博 |

### 2.4 用户使用场景

```
📱 场景 1：早通勤（8:00-9:00）
  打开 → 领取离线收益（修为+金币+装备）→ 看看悟空升了几级 → 关掉
  时长：2 分钟

📱 场景 2：午休（12:00-12:30）
  打开 → 手动推几关取经进度 → 挑战一次 Boss 战 → 安排挂机策略
  时长：10 分钟

📱 场景 3：摸鱼/等车
  打开 → 强化装备 → 点几下修炼点击 → 看看技能树要点什么
  时长：3-5 分钟

📱 场景 4：睡前（22:00-23:00）
  打开 → 推主线剧情（西游故事）→ 打今天的 Boss → 设置离线策略 → 睡觉
  时长：15 分钟

🌙 场景 5：睡觉中
  游戏在后台或关闭 → 悟空自动修炼 → 离线收益积累中
  时长：8 小时（自动）
```

## 3. 竞品分析

### 3.1 直接竞品（西游题材）

| 产品 | 类型 | DAU | 优势 | 劣势 |
|------|------|-----|------|------|
| 梦幻西游手游 | MMO | 千万 | 品牌+社交 | 太重，P2W |
| 大话西游手游 | 回合 RPG | 百万 | IP 强 | 需肝，同质化 |
| 少年西游记 | 卡牌放置 | 十万 | 放置+卡牌 | 换皮感强 |
| 黑神话悟空 | 单机 3A | — | 品质天花板 | 不是手游，非 idle |
| 放置西游类 | 放置 | 万级 | 轻度 | 品质差，换皮 |

**我们的机会**：西游题材玩家对「又一个卡牌/MMO」已疲劳。纯文字形态是全新体验，没有审美疲劳。

### 3.2 间接竞品（文字/Idle Game）

| 产品 | 类型 | 启示 |
|------|------|------|
| A Dark Room | 文字生存 | 纯文字可登顶 App Store，证明品类天花板 |
| Candy Box 1/2 | ASCII 增量 | 符号+数值有独立受众 |
| Universal Paperclips | 文字增量 | 病毒传播案例 |
| Kittens Game | 文字放置 | 长线留存标杆 |
| Idle Slayer | 像素放置 | 参考增长曲线 |
| Melvor Idle | RPG 放置 | 深度养成+RuneScape 怀旧成功 |

### 3.3 竞品对比表

| 维度 | 梦幻西游 | 少年西游记 | A Dark Room | Kittens Game | **西游 Idle v2.0** |
|------|---------|-----------|-------------|-------------|-------------------|
| 包体 | 3GB | 500MB | 1MB | 500KB | **~150KB** |
| 付费 | 重氪 | 中氪 | ¥6 买断 | 免费 | **免费开源** |
| 时间 | 重度 | 中度 | 轻度 | 轻度 | **轻度** |
| IP深度 | 高 | 低 | 无 | 无 | **高（81难还原）** |
| 美术 | AAA | 2D 卡牌 | 零 | 零 | **零（差异化）** |
| 养成深度 | 极深 | 中 | 浅 | 深 | **深** |
| 叙事 | 弱 | 弱 | 中 | 无 | **强（主线+支线）** |

### 3.4 机会点

1. **品类空白** — 没有「完整西游叙事 + Idle + 纯文字」的产品
2. **反卷审美** — 在所有人卷美术的时代，零图片是最强差异化
3. **开源信任** — 完全透明，没有暗箱，玩家信任度高
4. **极低门槛** — 150KB 秒开，不用下载，浏览器直接玩
5. **文化输出** — 英文版可成为海外用户了解西游记的入口

## 6. 品牌调性

### 6.1 视觉风格

| 维度 | 方向 |
|------|------|
| 核心 | 纯文字 + Unicode 符号 + Emoji |
| 色调 | 深色背景 + 品质色彩系统（⬜🟢🔵🟣🟡🔴） |
| 排版 | 等宽字体 + 框线字符 + 留白 |
| 感觉 | Terminal 美学 × 古典东方意境 |
| 禁区 | 不用图片、不用像素画、不用 3D，保持「零图片」纯粹性 |

### 6.2 文案调性

**主调性：轻古风 + 幽默 + 极客感**

| 场景 | 调性 | 示例 |
|------|------|------|
| 游戏内文案 | 半文言+白话混合 | 「汝之修为已至瓶颈，是否突破？」 |
| 社交媒体 | 轻松幽默 | 「悟空替你打了一晚上妖怪，你就说感不感动」 |
| 技术社区 | 极客+真诚 | 「150KB 做了个有 81 关的 RPG，是一种什么体验」 |
| 公告 | 简洁有力 | 「v2.0 上线。九九八十一难，从长安出发。」 |

**禁区**：
- 不用二次元/萌系语言（不符合西游IP调性）
- 不用网游套路话术（「充值送 SSR」之类）
- 不过度卖惨/营销焦虑

### 6.3 社交媒体人设

**名称**：西游 Idle 官方  
**人格**：一个正在西天取经路上的独立开发者

| 维度 | 描述 |
|------|------|
| 口吻 | 第一人称「我们」，偶尔用悟空/师徒视角 |
| 态度 | 真诚、不端着、有自嘲精神 |
| 互动 | 100% 回复，把用户当朋友 |
| 梗 | 西游记梗 + 程序员梗 + 放置游戏梗 |
| 禁止 | 不撕、不踩竞品、不画饼 |

**人设示例帖**：
```
今天修了一个 Bug：悟空离线打怪打到了第 82 难。

等等，一共才 81 难。

[已修复]
```

```
有用户反馈说「挂了一晚上才到第 5 难，太慢了」

师兄，取经本来就是慢功夫啊 🙏

（但我们还是偷偷调快了 20%）
```
