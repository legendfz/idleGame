## 📝 文案 2：玩法预告

**标题**：「不只是挂机——v2.0 的战斗，需要你的拇指」

**正文**：

v1.x 的核心是放置。关掉手机，悟空替你打怪，你只需要回来收菜。

v2.0 不一样了。

Boss 战变成了限时挑战。每个 Boss 有专属技能——白骨精三变、红孩儿三昧真火、牛魔王铁扇烈焰。屏幕上出现 ⚠️ 预警，你有 3 秒时间决定：闪避还是硬吃？

挂机的时候，悟空自动修炼，修为每秒增长。但当你主动点击，产出暴涨 10 倍。连续点击 10 次触发「连击加速」，5 秒内修炼速度翻倍。还有 5% 概率触发暴击——50 倍产出，金光一闪。

放置是底线，操作是上限。你可以佛系修仙，也可以疯狂点击卷到飞起。

```
📱 挂机模式：修为 +100/秒
👆 点击模式：修为 +1,000/次
💥 暴击点击：修为 +5,000/次（5%概率）
🔥 连击加速：全部×2，持续5秒
```

你选哪种修行方式？

## 📝 文案 3：数值深度

**标题**：「14 个境界 × 81 难 × 6 装备槽 = 一辈子玩不完」

**正文**：

做放置游戏最怕什么？玩两天就见底了。

v2.0 的数值深度，是为了让你「永远差一点」。

境界系统从 v1 的 10 级扩展到 14 级。从凡人到圣人，每个大境界 9 个小层级（九九之数），每次突破需要修为+材料+试炼关卡。不是点一下就升级——你得真的去打。

```
凡人 → 练气 → 筑基 → 金丹 → 元婴
→ 化神 → 渡劫 → 地仙 → 天仙
→ 金仙 → 太乙 → 大罗 → 混元 → 圣人
```

装备系统：5 个角色 × 6 个槽位 = 30 件装备要凑。每件有品质（⬜→🔴六级）、强化（+1→+15）、精炼。一件完美鸿蒙 +15 武器，可能需要吃掉上百件装备。

然后是转世。通关 81 难之后，「如来点化」，一切归零重来——但带着永久加成。每次转世都更强，每次都有新发现。

这不是两天见底的游戏。这是一条修行之路。

## 📝 文案 4：角色介绍

**标题**：「师徒五人，各有所长——你 main 谁？」

**正文**：

v2.0 的取经团队终于齐了。五个角色，五种玩法风格。

🧘 **唐僧** — 领袖/辅助
诵经加持全队，修炼速度 +15%。不能打，但没他就没有「取经」这件事。

🐒 **孙悟空** — 主 DPS
七十二变，暴击率 +10%，暴击伤害 +50%。三条技能树：棒法精通（肉搏），七十二变（变化），仙丹体质（修炼）。怎么点，你决定。

🐷 **猪八戒** — 肉盾/采集
食量大，金币掉落 +25%。前排扛伤害，顺便帮你捡钱。

🏜️ **沙悟净** — 辅助/负重
背包容量 +50%。沉默寡言但靠得住。装备党的最爱。

🐴 **白龙马** — 坐骑/加速
取经进度推进速度 +20%。虽然不参战，但没有他你走不快。

每个角色可以切换为主控。选悟空就是暴力输出，选唐僧就是全队增益，选八戒就是金币狂魔。

## 🎮 互动 1：境界猜猜猜

### 玩法说明
公布 v2.0 的 14 个修炼境界名称（打乱顺序），让玩家猜正确排列顺序。

### 参与流程

1. **发布帖**：公布 14 个境界名称（乱序排列），配文：「v2.0 有 14 个修炼境界，你能猜对排列顺序吗？」

```
以下 14 个境界，从低到高排列：

太乙 · 练气 · 凡人 · 金仙 · 化神 · 圣人
混元 · 筑基 · 天仙 · 大罗 · 渡劫 · 金丹
地仙 · 元婴

请在评论区写出你认为正确的顺序！
```

2. **参与方式**：评论区直接回复排列顺序
3. **公布答案**：48 小时后公布正确顺序
4. **结果互动**：统计猜对人数，「全对率」作为二次传播素材

### 奖励机制
- 完全正确：Discord 专属「先知」角色 + v2.0 上线后游戏内称号「天机洞悉」
- 猜对 10 个以上：Discord 「修仙达人」角色
- 参与即送：v2.0 上线提醒资格

### 适用渠道
Discord / QQ 群 / TapTap / Reddit

## 🎮 互动 2：组队招募

### 玩法说明
v2.0 有 5 个可选主控角色。让玩家提前选择自己的「main」角色，组建虚拟取经队伍。

### 参与流程

1. **发布帖**：

```
📢 v2.0 取经团队招募令

v2.0 的师徒五人各有所长，你会 main 谁？

🧘 唐僧 — 全队增益型，辅助流
🐒 悟空 — 暴力输出，暴击流
🐷 八戒 — 金币采集，富豪流
🏜️ 沙僧 — 背包扩容，囤货流
🐴 白龙马 — 推图加速，速通流

回复你的选择 + 理由！
```

2. **参与方式**：评论回复角色 emoji + 理由
3. **数据收集**：统计各角色选择比例
4. **结果发布**：制作角色人气排行榜，二次发帖

```
📊 取经团队人气榜
🐒 悟空  ████████████  47%
🐷 八戒  ██████       23%
🧘 唐僧  ████         15%
🏜️ 沙僧  ███          10%
🐴 白龙马 ██            5%
```

### 奖励机制
- 最冷门角色支持者：Discord 「独行侠」角色（选最少人选的角色）
- 最佳理由奖（3 名）：由社区投票选出，获 Discord 「取经先锋」角色
- 所有参与者：上线后该角色初始经验 +10% 加成码（需 CTO 配合）

### 适用渠道

## 🎮 互动 3：挑战预言

### 玩法说明
让玩家预测自己在 v2.0 上线后的游戏进度。上线一周后对比实际成绩，看谁预言最准。

### 参与流程

1. **发布帖**：

```
🔮 v2.0 挑战预言

v2.0 上线后第一周，你觉得自己能做到什么？

写下你的预言（选 1-3 项）：

⏱️ 时间类
  □ 第一天通关第一章（1-9 难）
  □ 三天内打到第 27 难
  □ 一周内突破金丹境界

⚔️ 战斗类
  □ 首日击败寅将军（第一章 Boss）
  □ 三天内挑战白骨精
  □ 一周内单挑牛魔王

💎 收集类
  □ 第一天获得第一件仙品装备
  □ 三天内收齐四个队友
  □ 一周内拥有一件混沌装备

💀 硬核类
  □ 一周内通关全部 81 难
  □ 一周内第一次转世
  □ 无氪通关（反正本来就免费 😂）

回复你的预言编号！
上线一周后，我们来验证谁是预言家 🔮
```

2. **参与方式**：评论选择 1-3 项预言
3. **验证阶段**：v2.0 上线一周后，发布验证帖，让玩家自行对照
4. **结果互动**：收集「预言成真」和「预言翻车」的截图，制作趣味对比帖

### 奖励机制
- 预言全部命中：Discord 「先知」角色 + 游戏内称号「未卜先知」
- 预言全部翻车：Discord 「乐观主义者」角色（翻车也有奖，鼓励参与）
- 最离谱预言奖：社区投票选出，获「天马行空」角色

### 适用渠道
Discord / QQ 群 / TapTap / Reddit
