# 🎮 西游·悟空传 v2.1 更新说明

> 发布日期：2026-03-02

道友们，v2.1 补丁更新来了！本次更新聚焦 Bug 修复和体验优化，让修仙之路更顺畅。

---

## 🐛 Bug 修复（8 项）

- 修复：境界突破不检查材料消耗，导致可无限突破 → 现在突破前严格校验并扣除所需材料
- 修复：修为需求公式与设计偏差达 42%，前期过快后期过慢 → 已按设计文档校准全部 14 个境界的修为曲线
- 修复：修炼速度未计入装备加成 → 穿戴装备后修炼速度正确提升
- 修复：角色被动技能未接入战斗/修炼计算 → 角色被动效果现已正常生效
- 修复：境界突破后解锁内容未显示 → 突破时弹出解锁提示（自动攻击、装备栏、转世等）
- 修复：离线收益弹窗未集成 → 回归游戏时展示离线期间获得的修为和金币
- 修复：超过 8 小时离线的回归奖励 bonus 未生效 → 离线超 8h 额外 +10% 修为奖励 🎁
- 修复：境界突破使用 alert 弹窗阻塞游戏 → 改为非阻塞 Toast 提示，体验更流畅
