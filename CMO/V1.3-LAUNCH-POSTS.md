I've been building a text/emoji-only idle RPG (zero images) based on Chinese mythology. v1.3 adds boss battles with real-time interaction — a skill warning system where you tap to dodge within a 3-second window.

Play: https://legendfz.github.io/idleGame/
Source: https://github.com/legendfz/idleGame

Interesting design challenge this update: making boss fights feel engaging in a pure-text UI. The solution was a warning → countdown → tap-to-dodge pattern. Bosses also have phase transitions (enrage at <50% HP) that change the text layout to convey urgency.

10 dungeons based on Journey to the West locations, 28 achievements with permanent bonuses, local leaderboards.

React + TypeScript + Zustand. PWA, offline-capable, ~150KB gzipped. All state in localStorage.
