## 应急预案

### 🔴 P0 — 游戏无法访问

| 症状 | 原因 | 应对 |
|------|------|------|
| 链接 404 | GitHub Pages 部署失败 | CTO 立即检查 Actions 日志，手动触发部署 |
| 加载白屏 | JS 错误 | CTO 检查 console，hotfix + 重新部署 |
| 移动端异常 | CSS/viewport 问题 | CTO 修复，10 分钟内上线 |

**营销侧动作**：暂停未发出的帖子，已发帖子编辑置顶「修复中」，社区群告知。

### 🟡 P1 — 存档丢失

| 症状 | 原因 | 应对 |
|------|------|------|
| 刷新后进度归零 | localStorage 写入失败 | CTO 排查，紧急 patch |

**营销侧动作**：社区帖回复致歉 + 说明修复进度。

### 🟡 P2 — 帖子被删/限流

| 症状 | 原因 | 应对 |
|------|------|------|
| Reddit 帖被 automod 删除 | 新号/链接触发规则 | 联系 mod，按社区规则调整重发 |
| HN 帖下沉 | 标题不够吸引 | 不补发，转投其他渠道 |
| V2EX 被移节点 | 发错节点 | 接受移动，不争论 |

### 🟢 P3 — 流量超预期

| 症状 | 应对 |
|------|------|
| GitHub Pages 限流 | 静态站理论上无限流量，不太可能。备选：Vercel/Netlify 镜像 |
| 评论/反馈太多回不过来 | 优先回复负面反馈和 bug 报告 |
