## 1. 三大功能宣传文案

### 🔴 Feature 1：鸿蒙品质 — 6 级装备体系

#### 核心卖点
> 混沌之上，还有鸿蒙。10 件混沌精炼，10% 概率，一击入魂。

#### 长文案（社区帖/开发日志）

品质体系从 5 级扩展到 6 级，新增最高品质「🔴 鸿蒙」。

```
装备品质进化：
⬜ 凡品 → 🟢 灵品 → 🔵 仙品 → 🟣 神品 → 🟡 混沌 → 🔴 鸿蒙（NEW!）

鸿蒙获取：
  🟡×10 → 精炼 → 10% 概率 → 🔴
  失败不消耗装备，获得「鸿蒙碎片」×1
  碎片 ×10 = 保底兑换（100% 成功）

鸿蒙专属强化：+11 ~ +15
  +15 满级特殊标记：🔴 鸿蒙·极 +15 ✦
```

80 倍基础倍率。一件鸿蒙顶两件混沌。但 10% 的精炼概率，让每次成功都值得截图发群里。

#### 短文案
- 「⬜🟢🔵🟣🟡🔴 — 六色齐了，鸿蒙降世」
- 「10% 精炼成功率，100% 的快乐」
- 「混沌之上，还有鸿蒙。你的悟空配得上最强装备。」

---

### 🌙 Feature 2：离线掉落 — 睡觉也能捡装备

#### 核心卖点
> 关掉手机去睡觉，早上回来不只有灵石，还有装备。

#### 长文案

v1.0 的离线收益只有灵石和经验。v1.1 加入了**离线装备掉落**——你的悟空在你睡觉时不只打怪，还会捡装备。

```
🌙 离线 8 小时报告：
  💰 灵石 +284,500
  ✨ 经验 +127,300
  📦 装备 ×3（NEW!）
    🟢 灵品·破妖刀
    🔵 仙品·龙鳞甲
    🟣 神品·紫金铃 ← ！
```

掉落品质基于你当前关卡和境界，高境界解锁高品质掉落池。离线时间越长，掉落次数越多（上限 24h）。

#### 短文案
- 「睡觉也能捡神装，这才是真正的放置游戏」
- 「离线 8 小时，回来 3 件装备。你的悟空比你勤快。」
- 「v1.1 离线不只产灵石了 📦」

---

### 📬 Feature 3：游戏内反馈系统

#### 核心卖点
> 想吐槽？想要新功能？游戏里一键反馈，开发者真的会看。

#### 长文案

v1.1 在设置页新增了反馈入口。点一下就能报 Bug 或提建议，零注册零门槛。

```
⚙️ 设置
  ├── 📬 反馈
  │   ├── 🐛 报告 Bug
  │   ├── 💡 建议功能
  │   ├── 💬 加入社区
  │   └── ⭐ 给个好评
```

所有反馈直达 GitHub Issues，开发团队 24 小时内回复。你的每一条建议都可能出现在下个版本的 changelog 里。

#### 短文案
- 「你的反馈 = 我们的 roadmap」
- 「游戏里点一下就能给开发者提建议，独立游戏就该这样」
