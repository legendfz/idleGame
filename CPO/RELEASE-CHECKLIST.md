## 2. 性能

- [ ] 自动战斗 tick 无内存泄漏（长时间运行 1h+ 内存稳定）
- [ ] 战斗日志上限 30 条，不无限增长
- [ ] 存档 JSON 大小合理（<500KB，背包满 50 件时）
- [ ] 点击响应 <100ms，无卡顿
- [ ] 页面首次加载 <3 秒（4G 网络）
