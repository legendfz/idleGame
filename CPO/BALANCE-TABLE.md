## 1. 角色属性成长曲线（Lv.1–100）

### 1.1 基础公式

| 属性 | 公式 | 说明 |
|------|------|------|
| HP | `floor(100 + level * 18 + level^1.8 * 0.5)` | 生存能力，前期线性后期加速 |
| 攻击 | `floor(10 + level * 3.2 + level^1.6 * 0.15)` | 自动战斗伤害基础 |
| 防御 | `floor(5 + level * 1.5 + level^1.4 * 0.1)` | 减伤，硬上限 80% |
| 速度 | `floor(10 + level * 0.5 + level^1.2 * 0.05)` | 影响攻击间隔 |

### 1.3 升级经验公式
- 升级所需经验 = `floor(10 * level^1.5)`
- 累计经验 = `Σ(floor(10 * i^1.5), i=1..level)`

### 4.3 收支平衡校验（Lv.30 为例）

**收入侧**（挂关卡 30，1 小时）：
- 每关 ~90 秒，1 小时 ≈ 40 关
- 每关灵石 ≈ 10 × 1.12^30 × 11 波 ≈ 3,290/关
- 1 小时收入 ≈ **131,600 灵石**

**支出侧**（Lv.30 升到 31）：
- 升级经验 = 1,643（通过打怪获取，不消耗灵石）
- 属性升级 ≈ 10 × 30^1.3 ≈ 750 灵石
- 装备强化 +5 → +6 ≈ 80 × 6 × 10 = 4,800 灵石

**结论**：收入远大于单次支出 → 玩家有持续成长感 ✅
灵石主要消耗点：装备精炼（大额）、后期属性升级（指数增长）

## 6. 战斗伤害公式汇总

### 6.1 自动攻击伤害
```
基础伤害 = max(1, 玩家攻击 - 敌人防御)
暴击判定 = rand() < 暴击率 ? 基础伤害 × (1.5 + 暴伤%) : 基础伤害
最终伤害 = 暴击判定 × 技能倍率 × (1 + 队友攻击加成%)
```

### 6.2 点击伤害
```
点击伤害 = max(1, 点击力属性)
最终点击 = 点击伤害 × (1 + 法宝点击加成%) × (1 + 队友点击加成%)
```

### 6.3 DPS 估算
```
自动 DPS = 最终伤害 / 攻击间隔
点击 DPS = 最终点击 × 点击频率（假设 3 次/秒）
总 DPS = 自动 DPS + 点击 DPS + 技能 DPS
```

### 6.4 玩家受伤
```
受伤 = max(1, 敌人攻击 - 玩家防御) × (1 - 减伤%) × (1 - 闪避?)
```

玩家死亡 → 复活选项（免费 1 次/关，之后看广告或消耗回复丹）
