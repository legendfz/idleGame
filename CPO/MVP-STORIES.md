# MVP 用户故事 (User Stories)

**版本**：v1.0  
**日期**：2026-03-01  
**来源**：PRD v1.0 Phase 1 范围

---

## P0 — 核心循环（必须有）

### US-01: 自动战斗
**作为** 玩家，**我想** 进入游戏后自动攻击当前敌人，**以便** 我可以挂机获得收益。

**验收标准**：
- [ ] 每秒自动攻击一次，造成 `攻击力 - 敌防御（最低1）` 伤害
- [ ] 敌人血条实时更新
- [ ] 战斗日志显示每次攻击的伤害数字
- [ ] 暴击时显示特殊标记和加成伤害

### US-02: 点击攻击
**作为** 玩家，**我想** 点击屏幕造成额外伤害，**以便** 主动加速战斗。

**验收标准**：
- [ ] 点击战斗区域触发一次额外攻击
- [ ] 点击伤害独立于自动攻击
- [ ] 战斗日志显示"👆 点击攻击"
- [ ] 有点击视觉反馈（缩放动画）

### US-03: 击杀掉落
**作为** 玩家，**我想** 击败敌人后获得灵石和经验，**以便** 我可以升级角色。

**验收标准**：
- [ ] 敌人HP归零时显示击败动画/文字
- [ ] 自动获得灵石和经验，数量在战斗日志中显示
- [ ] Boss 有概率掉落蟠桃
- [ ] 顶栏资源数字实时更新

### US-04: 波次推进
**作为** 玩家，**我想** 击败当前敌人后自动进入下一波，**以便** 持续推进关卡。

**验收标准**：
- [ ] 每关 10 波小怪 + 1 Boss
- [ ] 小怪随机从当前章节怪物池抽取
- [ ] Boss 出现时有醒目提示
- [ ] Boss 击败后自动进入下一关
- [ ] 章节最后一关 Boss 击败后进入下一章

### US-05: 角色升级
**作为** 玩家，**我想** 经验满后自动升级，**以便** 属性持续增长。

**验收标准**：
- [ ] 经验达到 `floor(10 * level^1.5)` 时自动升级
- [ ] 升级增加攻击力和血量
- [ ] 升级时显示醒目提示（升级！Lv.X → Y）
- [ ] 可连续升级（经验溢出时）
- [ ] 点击伤害随等级增长

## P0 — 境界系统

### US-06: 境界突破
**作为** 玩家，**我想** 满足条件后突破境界，**以便** 获得属性倍增和新功能。

**验收标准**：
- [ ] 队伍页面显示当前境界和下一境界要求
- [ ] 显示所需等级和蟠桃数量
- [ ] 满足条件后"突破"按钮可点击
- [ ] 突破成功：扣除蟠桃，属性 ×1.5，境界名更新
- [ ] 战斗日志显示突破成功消息
- [ ] 前5个境界数据完整（灵猴初醒→元婴）

## P0 — 存档系统

### US-07: 自动存档
**作为** 玩家，**我想** 游戏自动保存进度，**以便** 关闭浏览器后不会丢失。

**验收标准**：
- [ ] 每 30 秒自动存档到 localStorage
- [ ] 存档包含：玩家属性、当前关卡、资源、游戏时间
- [ ] 存档含版本号用于未来迁移

### US-08: 离线收益
**作为** 玩家，**我想** 离线期间也能积累资源，**以便** 回来时有惊喜。

**验收标准**：
- [ ] 打开游戏时计算离线时长（最大24小时）
- [ ] 离线收益 = 在线DPS × 50% × 离线时长
- [ ] 显示离线收益报告弹窗（时长、灵石、经验）
- [ ] 点击"领取"关闭弹窗
- [ ] 离线时间不足1分钟不显示报告

## P1 — 章节内容

### US-09: 花果山章节（第1章）
**作为** 玩家，**我想** 从花果山开始冒险，**以便** 体验石猴出世的故事。

**验收标准**：
- [ ] 50 关，等级范围 1-30
- [ ] 3种小怪（野狼、石头精、树妖）
- [ ] Boss：混世魔王
- [ ] 怪物数值按关卡递增（×1.12^n）

### US-10: 龙宫章节（第2章）
**验收标准**：
- [ ] 60 关，等级范围 30-80
- [ ] 3种小怪（虾兵、蟹将、巡海夜叉）
- [ ] Boss：龙王三太子
- [ ] 第1章通关后解锁

### US-11: 天庭章节（第3章）
**验收标准**：
- [ ] 100 关，等级范围 80-200
- [ ] 3种小怪（天兵、天将、巨灵神）
- [ ] Boss：二郎神
- [ ] 第2章通关后解锁

## P1 — UI/UX

### US-12: 顶栏信息
**验收标准**：
- [ ] 显示当前场景名、关卡数
- [ ] 显示灵石、蟠桃、等级、境界名
- [ ] 常驻固定在顶部

### US-13: 底部导航
**验收标准**：
- [ ] 5个 Tab：战斗、队伍、旅途、背包、更多
- [ ] 当前 Tab 高亮
- [ ] 点击切换页面内容

### US-14: 旅途地图
**验收标准**：
- [ ] 展示所有章节列表
- [ ] ✅ 已通关 / 🔥 进行中 / 🔒 未解锁
- [ ] 当前章节显示进度（x/total）

### US-15: 设置页面
**验收标准**：
- [ ] 显示总游戏时间
- [ ] 手动存档按钮
- [ ] 重置游戏按钮（需确认）
