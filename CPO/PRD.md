## 1. 产品概述

纯文字+符号的竖屏 Idle Game，无图片资源。玩家扮演孙悟空，从花果山小猴子成长为斗战胜佛。核心体验：**挂机成长 + 章节推图 + 队友收集**。

### 1.1 设计原则
- **纯文字渲染**：所有 UI 用 Unicode 符号 + Emoji + ASCII art
- **竖屏优先**：移动端单手操作
- **离线友好**：关掉也在成长，回来有惊喜
- **数值爽感**：大数字、频繁升级、明确的进度感

## 2. 核心玩法循环

```
┌─────────────────────────────────────────┐
│  点击/自动战斗 → 获得经验+灵石 → 升级  │
│       ↑                          ↓      │
│  解锁新区域  ← 突破境界 ← 修炼/强化    │
└─────────────────────────────────────────┘
```

### 2.1 战斗系统（核心循环）

**自动战斗**：
- 进入关卡后自动攻击当前敌人，每 1 秒造成一次伤害
- 伤害 = 攻击力 - 敌人防御（最低 1）
- 敌人被击败 → 掉落灵石 + 经验 → 自动进入下一波
- 每关 10 波小怪 + 1 波 Boss

**主动点击**：
- 点击屏幕 = 额外攻击一次（点击伤害独立计算）
- 点击伤害初始 = 1，可通过升级/法宝提升
- 连击奖励：连续快速点击 10 次触发「金箍棒横扫」，全屏 AOE

**战斗显示**（纯文字示例）：
```
═══ 花果山·第3关 [7/10] ═══
🐺 灰狼精  ████████░░  HP: 847/1200
⚔️ 自动攻击  -156
👆 点击攻击  -23
💰 灵石: 12,847  ✨ 经验: 4,521
```

### 2.2 升级循环

| 资源 | 获取途径 | 用途 |
|------|---------|------|
| 灵石 | 击杀怪物、离线收益 | 升级属性、购买物品 |
| 经验 | 击杀怪物、完成任务 | 角色升级 |
| 修为 | 境界突破后积累 | 解锁高级技能 |
| 蟠桃 | Boss 掉落（稀有） | 突破境界 |

## 3. 成长系统

### 3.1 等级系统
- 等级上限：9999
- 每次升级：+基础攻击、+基础血量
- 升级所需经验 = `floor(10 * level^1.5)`
- 每 10 级解锁一个属性点自由分配

### 3.2 境界系统（核心成长里程碑）

| 境界 | 等级要求 | 蟠桃 | 解锁内容 |
|------|---------|------|---------|
| 灵猴初醒 | 1 | — | 基础攻击 |
| 通灵 | 10 | 5 | 自动战斗 |
| 炼气 | 30 | 20 | 技能栏×1 |
| 筑基 | 60 | 80 | 队友栏×1 |
| 金丹 | 100 | 200 | 法宝栏×1 |
| 元婴 | 200 | 500 | 离线收益×2 |
| 化神 | 350 | 1200 | 技能栏×2 |
| 大乘 | 500 | 3000 | 转世系统（Prestige） |
| 齐天大圣 | 800 | 8000 | 全属性×5 |
| 斗战胜佛 | 1500 | 20000 | 终极形态 |

**突破玩法**：
- 达到等级 + 蟠桃要求后，进入「心魔试炼」（限时 Boss 战）
- 成功 → 境界提升，属性倍增
- 失败 → 保留材料，可重试

### 3.3 属性系统

| 属性 | 效果 |
|------|------|
| 攻击 | 自动战斗伤害 |
| 血量 | 生存能力 |
| 速度 | 攻击间隔缩短 |
| 暴击 | 暴击率（%） |
| 暴伤 | 暴击伤害倍率 |
| 点击力 | 点击伤害 |

## 4. 离线收益机制

### 4.1 基础离线收益
- 离线时自动战斗**最后通关的关卡**
- 离线效率 = 在线 DPS 的 **50%**（可通过境界/法宝提升至 100%）
- 离线时间上限：**24 小时**（VIP/道具可延长至 48h）

### 4.2 离线结算

回归时显示：
```
═══ 🌙 离线修炼报告 ═══
⏱️ 离线时长：8小时23分
⚔️ 击败怪物：12,847 只
💰 获得灵石：1,284,700
✨ 获得经验：892,340
🍑 获得蟠桃：3
📦 获得装备：铁棒×1

[✅ 领取] [📺 看广告×2]
```

### 4.3 离线加速
- 「定身符」道具：暂停离线时间消耗（攒更长离线时间）
- 「时光葫芦」：一次性增加 4 小时离线时间
- 看广告：离线收益 ×2（单次）

### 6.2 收服妖怪

- 部分 Boss 击败后有概率「收服」（基础 5%，可通过「紧箍咒」提升）
- 收服的妖怪作为额外队友，提供被动加成
- 队友上阵上限：主角 + 4 队友（师徒占固定位）
- 妖怪队友可升级（消耗灵石），提升被动效果

## 8. 转世系统（Prestige）

### 8.1 触发条件
- 到达第6章「五行山」
- 境界达到「大乘」

### 8.2 转世效果
- 等级重置为 1，章节重置为第1章
- **保留**：境界、法宝、收服的妖怪
- **获得**：「功德」— 永久全属性加成
- 功德值 = `floor(等级 / 100) * (1 + 转世次数 * 0.1)`
- 每次转世后，成长速度加快，更快推到更远的章节

### 8.3 转世奖励层
| 转世次数 | 额外奖励 |
|---------|---------|
| 1 | 起始灵石 ×10 |
| 3 | 自动战斗初始解锁 |
| 5 | 起始境界「炼气」 |
| 10 | 解锁「真·齐天大圣」称号 |

## 附录：数值平衡参考

### 怪物 HP 增长
- 关卡 n 的 Boss HP = `floor(100 * 1.15^n)`
- 小怪 HP = Boss HP × 0.2

### 灵石掉落
- 关卡 n 每波灵石 = `floor(10 * 1.12^n)`

### 预期节奏
- 第1章（50关）：~2 小时主动 / ~6 小时挂机
- 第2章（80关）：~4 小时主动 / ~14 小时挂机
- 首次转世：~3-5 天
