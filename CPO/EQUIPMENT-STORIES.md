# 装备系统用户故事

**版本**：v1.0
**日期**：2026-03-01
**依据**：PRD.md §7 装备/法宝系统，BALANCE-TABLE.md §2 装备数值体系

## EQ-01: 装备掉落与拾取

**作为** 玩家，**我想** 击败怪物后有概率获得装备，**以便** 提升战斗力。

**验收标准**：
- [ ] 小怪掉落装备概率 5%，Boss 掉落概率 30%，Boss 首通 100%
- [ ] 掉落品质按权重随机：⬜70%/🟢25%/🔵4.5%/🟣0.5%（小怪）；Boss 概率上移
- [ ] 装备等级 = 关卡推荐等级 ± rand(3)
- [ ] 掉落时战斗日志显示「📦 获得 [品质色] 装备名」
- [ ] 装备自动进入背包

---

## EQ-02: 装备穿戴与卸下

**作为** 玩家，**我想** 在背包中穿戴或更换装备，**以便** 立即生效属性加成。

**验收标准**：
- [ ] 4 个栏位：武器（第3章解锁）、护甲（第3章）、法宝（金丹境界）、饰品（第5章）
- [ ] 未解锁栏位显示 🔒 和解锁条件
- [ ] 点击背包装备 → 显示属性对比（当前 vs 新）
- [ ] 穿戴后属性立即刷新，战斗面板实时更新
- [ ] 卸下装备回到背包

---

## EQ-03: 装备强化（升级）

**作为** 玩家，**我想** 消耗灵石强化装备，**以便** 提升装备基础属性。

**验收标准**：
- [ ] 强化等级 +1 ~ +10，每级提升基础属性 10%
- [ ] 强化费用 = `floor(基础属性 × 强化等级 × 10)` 灵石
- [ ] +1 ~ +5：100% 成功
- [ ] +6 ~ +7：80% 成功，失败不掉级，消耗灵石 50%
- [ ] +8 ~ +9：60% 成功，失败不掉级，消耗灵石 50%
- [ ] +10：40% 成功，失败不掉级，消耗灵石 50%
- [ ] 成功显示「✨ 强化成功！+N → +N+1」，失败显示「💔 强化失败」
- [ ] 灵石不足时按钮置灰，显示差额

---

## EQ-04: 装备精炼（品质提升）

**作为** 玩家，**我想** 用多件同品质装备精炼出更高品质装备，**以便** 获得质的飞跃。

**验收标准**：
- [ ] 精炼规则：⬜→🟢（3件，100%）、🟢→🔵（5件，80%）、🔵→🟣（5件，50%）、🟣→🟡（8件，20%）
- [ ] 材料装备必须是同品质（不限类型）
- [ ] 精炼失败：不消耗装备，消耗灵石（精炼费 50%）
- [ ] 精炼费 = `floor(品质倍率 × 1000 × 目标品质等级)`
- [ ] 精炼成功：品质提升，基础属性按新品质倍率重算，保留强化等级
- [ ] 精炼 UI 显示所需材料数、成功率、费用

## EQ-05: 装备附魔

**作为** 玩家，**我想** 用仙石给装备附加随机词缀，**以便** 获得特殊效果。

**验收标准**：
- [ ] 每件装备最多 2 条附魔
- [ ] 每次附魔消耗 500 仙石
- [ ] 词缀从 8 种池中按权重随机：攻击%/HP%/暴击/暴伤/灵石掉落/经验/击杀回复/点击伤害
- [ ] 已有 2 条附魔时，新附魔替换玩家选择的一条
- [ ] 附魔前预览「将替换：[旧词缀]」，需确认
- [ ] 仙石不足时提示获取途径

---

## EQ-06: 装备分解

**作为** 玩家，**我想** 分解不需要的装备，**以便** 回收仙石和灵石。

**验收标准**：
- [ ] ⬜ 凡品：返还灵石（基础属性 × 5），无仙石
- [ ] 🟢 灵品：灵石 + 10 仙石
- [ ] 🔵 仙品：灵石 + 30 仙石
- [ ] 🟣 神品：灵石 + 80 仙石
- [ ] 🟡 混沌：灵石 + 200 仙石
- [ ] 强化过的装备额外返还 30% 强化投入灵石
- [ ] 支持批量分解（按品质筛选）
- [ ] 分解前显示回收预览，需确认
- [ ] 已穿戴的装备不可直接分解

## EQ-07: 标志性法宝获取

**作为** 玩家，**我想** 在剧情节点获得标志性法宝，**以便** 体验经典西游故事。

**验收标准**：
- [ ] 火眼金睛：第2章剧情自动获得，暴击率 +15%
- [ ] 筋斗云：第2章剧情自动获得，攻速 +20%
- [ ] 如意金箍棒：第3章剧情自动获得，攻击 +100%
- [ ] 定海神针：第3章隐藏关卡掉落，点击伤害 ×3
- [ ] 法宝获得时全屏特效文字 + 法宝说明弹窗
- [ ] 法宝不可分解、不可卸下（永久装备）
- [ ] 法宝可用灵石升级（独立升级线，费用 ×5）

## EQ-08: 背包管理

**作为** 玩家，**我想** 在背包中查看、排序和管理所有装备，**以便** 快速找到最优装备。

**验收标准**：
- [ ] 背包初始容量 50，可用灵石扩容（每次 +10，费用递增）
- [ ] 排序方式：品质 / 等级 / 攻击 / HP / 最新获得
- [ ] 筛选：按栏位类型（武器/护甲/法宝/饰品）、按品质
- [ ] 装备详情页显示：名称、品质、等级、基础属性、强化等级、附魔词缀
- [ ] 「一键穿戴最强」按钮：自动穿戴每个栏位属性最高的装备
- [ ] 背包满时提示「背包已满，请分解或扩容」

---

## EQ-09: 装备对比

**作为** 玩家，**我想** 对比两件装备的属性差异，**以便** 做出穿戴决策。

**验收标准**：
- [ ] 点击背包装备自动与已穿戴的同栏位装备对比
- [ ] 绿色 ▲ 显示提升项，红色 ▼ 显示下降项
- [ ] 对比包含：基础属性、强化加成、附魔词缀
- [ ] 显示总战力变化预估（+X / -X）
- [ ] 无同栏位装备时只显示装备详情
