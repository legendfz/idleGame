## 5. 配色与动效

### 5.1 新增颜色

| 用途 | 颜色 | Hex |
|------|------|-----|
| 副本可挑战边框 | 橙色 | `#ff9800` |
| 技能预警 | 红色闪烁 | `#ff4444` |
| 闪避按钮 | 亮蓝 | `#42a5f5` |
| 成就完成背景 | 暗绿 | `#1a3a1a` |
| 新纪录 | 金色 | `#f0c040` |
| 排行前三 | 金/银/铜 | `#ffd700` / `#c0c0c0` / `#cd7f32` |

### 5.2 新增动效

| 动效 | 场景 | 实现 |
|------|------|------|
| 脉冲闪烁 | 闪避按钮 | `@keyframes pulse { 0%,100% { transform: scale(1) } 50% { transform: scale(1.05) } }` |
| 红色闪烁 | 技能预警框 | `@keyframes warning { 0%,100% { border-color: #ff4444 } 50% { border-color: transparent } }` |
| 滑入 | Toast 通知 | `transform: translateY(-100%) → translateY(0)` 300ms ease-out |
| 进度填充 | 成就进度条 | `transition: width 0.5s ease` |
| 血条渐变 | Boss HP | 动态 `background: linear-gradient` 根据 HP 百分比 |
