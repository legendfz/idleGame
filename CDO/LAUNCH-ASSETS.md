## 截图清单

需截取 **6 张**关键画面，用于应用商店/社交媒体/落地页。

---

### 截图 1：主战斗界面（Hero Shot）

**用途**：应用商店首图、落地页首屏、社交媒体主推
**构图**：

```
要求：
- 玩家等级 Lv.15+，有一定数值积累
- 正在战斗中，敌人血条约 40~60%（视觉上有交战感）
- 战斗日志显示 2~3 条记录（含一条暴击 💥）
- 底部挂机收益区可见
- BottomNav 完整可见
```

**注意事项**：
- 确保金币/仙玉数字不为 0（展示有收益感）
- 暴击行颜色醒目

### 截图 3：角色/队伍界面

**用途**：商店截图第 3 张、展示角色养成深度

```
要求：
```

## 截图方法

由于是 Web 应用（已部署 GitHub Pages），建议：
1. Chrome DevTools → Device Mode → iPhone 14 Pro (390×844)
2. 手动玩到合适状态
3. `⌘+Shift+P` → "Capture screenshot"
4. 或使用 Playwright 自动化脚本截图
