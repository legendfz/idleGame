### 1.2 列表行规范

每个装备行 3 行结构：

```
行1：{类型ico} {品质前缀}{装备名}
行2：  {主属性ico}+{值}  {副属性ico}+{值}
行3：  {标签}                    Lv.{n}
```

- 品质前缀颜色 = 品质色（名称也染色）
- `[装备中]` 标签用 `--c-accent` 金色
- 已装备行左侧有 2px 金色竖线指示

### 1.4 排序规则

1. 已装备置顶
2. 品质降序（神品 > 仙品 > 极品 > 上品 > 良品 > 凡品）
3. 同品质按 Lv 降序
4. 同 Lv 按类型：武器 > 防具 > 法宝

### 2.3 对比面板规范

装备替换时显示属性对比：

```
┌─ 对比 ──────────────────────┐
│ {属性ico} {属性名}  {旧值} → {新值}  {差值} │
└─────────────────────────────┘
```

- 差值正数：`▲{n}` 绿色 `--c-success`
- 差值负数：`▼{n}` 红色 `--c-danger`
- 无变化：`─` 灰色

### 2.4 套装面板规范

仅当物品属于套装时显示：

```
┌─ 套装 ──────────────────────┐
│ 「{套装名}」 {已收集}/{总数}  │
│  {✅/○} {类型ico} {物品名}   │  × N 件
│ ──────────────────           │
│ {n}件：{效果}  {n}件：{效果}  │
└─────────────────────────────┘
```

- `✅` 已拥有件，正常文字色
- `○` 未拥有件，灰色 `--c-text-sub`
- 已激活套装效果：金色 `--c-accent`
- 未激活：灰色

### 2.5 底部操作按钮

| 场景 | 按钮组 |
|------|--------|
| 已装备 | `[ 🔨 强化 ]` `[ ❌ 卸下 ]` |
| 未装备 | `[ ✅ 装备 ]` `[ 🔨 强化 ]` `[ 🪙 卖出 ]` |
| 满级装备 | `[ 🔨 已满级 ]`(禁用) `[ ❌ 卸下 ]` |
| 卖出确认 | `确定卖出？ 获得 🪙{n}` `[ 确认 ]` `[ 取消 ]` |

### 3.5 CSS 动画规范

```css
/* 锤子摇摆 */
@keyframes hammer-swing {
  0%, 100% { transform: rotate(0deg); }
  25%      { transform: rotate(-15deg); }
  75%      { transform: rotate(15deg); }
}
.enhance-charging { animation: hammer-swing 600ms ease-in-out; }

/* 锤击下落 */
@keyframes hammer-strike {
  0%   { transform: translateY(-20px) rotate(-30deg); }
  100% { transform: translateY(0) rotate(0); }
}
.enhance-strike { animation: hammer-strike 200ms ease-in; }

/* 物品脉动 */
@keyframes item-pulse {
  0%, 100% { transform: scale(1); }
  50%      { transform: scale(1.02); }
}
.enhance-pulse { animation: item-pulse 300ms ease-in-out infinite; }

/* 物品震动（失败） */
@keyframes item-shake {
  0%, 100% { transform: translateX(0); }
  20%, 60% { transform: translateX(-4px); }
  40%, 80% { transform: translateX(4px); }
}
.enhance-fail { animation: item-shake 300ms ease; }

/* 粒子扩散 */
@keyframes sparkle-burst {
  0%   { transform: scale(0); opacity: 1; }
  100% { transform: scale(2); opacity: 0; }
}
.enhance-sparkle { animation: sparkle-burst 500ms ease-out forwards; }

/* 结果弹入 */
@keyframes result-pop {
  0%   { transform: scale(0.8); opacity: 0; }
  60%  { transform: scale(1.05); }
  100% { transform: scale(1); opacity: 1; }
}
.enhance-result { animation: result-pop 400ms cubic-bezier(0.34, 1.56, 0.64, 1); }
```
