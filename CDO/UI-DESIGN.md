## 1. 视觉风格定义

### 1.1 设计原则
- **纯文字+符号**，零图片资源
- 竖屏优先（移动端 Web）
- 信息密度适中，留白呼吸
- 用 Unicode/Emoji 符号建立视觉层次

### 1.2 符号体系

#### 角色符号
| 角色 | 符号 | 说明 |
|------|------|------|
| 悟空（玩家） | 🐒 | 主角 |
| 唐僧 | 🧘 | 师父/队友 |
| 八戒 | 🐷 | 队友 |
| 沙僧 | 🏔️ | 队友 |
| 白龙马 | 🐴 | 坐骑/队友 |
| 普通妖怪 | 👹 | 小怪 |
| 精英妖怪 | 👺 | 精英 |
| Boss 妖怪 | 🐉 | Boss |
| 神仙/NPC | ☁️ | 天庭/NPC |

#### 装备/法宝符号
| 类型 | 符号 | 示例 |
|------|------|------|
| 武器 | ⚔️ | 如意金箍棒 |

## 2. 竖屏排版布局

### 2.1 全局框架（自上而下）
```
┌──────────────────────┐
│ 📍 顶栏：场景 + 资源  │  ← 固定
├──────────────────────┤
│                      │
│                      │
│    主内容区域         │  ← 可滚动
│                      │
│                      │
├──────────────────────┤
│ 🔘 底部导航栏         │  ← 固定
└──────────────────────┘
```

### 2.2 顶栏（常驻）
```
─────────────────────────
📍 花果山·水帘洞    第3章
🪙 12,450  💠 38  ✨ Lv.15
─────────────────────────
```
- 第一行：当前场景 + 章节
- 第二行：核心资源速览

### 2.3 底部导航栏（5 Tab）
```
─────────────────────────
 🗡️    🐒    🗺️    📦    ⚙️
 战斗   队伍   旅途   背包   更多
─────────────────────────
```

| Tab | 图标 | 功能 |
|-----|------|------|
| 战斗 | 🗡️ | 当前战斗/挂机 |
| 队伍 | 🐒 | 角色详情/阵容 |
| 旅途 | 🗺️ | 章节地图/进度 |
| 背包 | 📦 | 物品/装备管理 |
| 更多 | ⚙️ | 设置/成就/商店 |

## 3. 信息架构

### 3.1 导航结构
```
App
├── 🗡️ 战斗（默认首页）
│   ├── 自动战斗面板
│   ├── 战斗日志
│   └── 挂机收益
├── 🐒 队伍
│   ├── 角色详情
│   ├── 装备管理
│   ├── 技能升级
│   └── 阵容编排
├── 🗺️ 旅途
│   ├── 章节列表
│   ├── 关卡详情
│   └── 剧情回顾
├── 📦 背包
│   ├── 装备
│   ├── 法宝
│   ├── 丹药
│   └── 材料
└── ⚙️ 更多
    ├── 🏪 商店
    ├── 🏆 成就
    ├── 📊 统计
    └── ⚙️ 设置
```

### 3.2 交互模式
- **底部 Tab**：主导航，5 个入口
- **顶部 Sub-Tab**：二级过滤（如背包分类）
- **列表点击**：进入详情弹层（半屏上拉）
- **左滑操作**：快捷操作（卖出/使用）
- **长按**：物品详情 tooltip

| 事件 | 表现 |
|------|------|
| 点击按钮 | 短暂文字加粗/放大 → 恢复 |

| 血条变化 | 条形从旧值滑动到新值（300ms ease-out） |
| 战斗日志 | 新行从底部推入，旧行上移 |

## 5. 色彩方案

### 5.1 暗色主题（默认）

以「东方水墨」为灵感，深底浅字：

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#1a1a2e` | 深靛蓝 |
| 卡片/面板 | `#16213e` | 略浅蓝 |
| 主文字 | `#e8e8e8` | 近白 |
| 次要文字 | `#8b8b9e` | 灰紫 |
| 强调色 | `#f0c040` | 金色（仙气） |
| 成功/正面 | `#4caf50` | 绿 |
| 危险/负面 | `#f44336` | 红 |
| 链接/可交互 | `#64b5f6` | 亮蓝 |
| 分隔线 | `#2a2a4a` | 暗线 |

### 5.2 亮色主题

以「宣纸」为灵感，浅底深字：

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#f5f0e8` | 宣纸黄 |
| 卡片/面板 | `#ffffff` | 纯白 |
| 主文字 | `#2c2c2c` | 深墨 |
| 次要文字 | `#7a7a7a` | 灰 |
| 强调色 | `#c49000` | 暗金 |
| 成功/正面 | `#388e3c` | 深绿 |
| 危险/负面 | `#d32f2f` | 深红 |
| 链接/可交互 | `#1976d2` | 蓝 |
| 分隔线 | `#d4cfc4` | 浅灰 |

### 5.3 字体建议
- 中文：系统默认（`system-ui`）
- 数字/符号：等宽字体（`monospace`）— 保证进度条对齐
- 字号：正文 14px，标题 16px，数字 15px

## 6. 响应式考虑

- **设计基准**：375px 宽（iPhone SE）
- **最大宽度**：480px（居中显示）
- **横屏**：不做特殊适配，保持竖屏布局
- **大屏（iPad/桌面）**：内容区居中，两侧留白
