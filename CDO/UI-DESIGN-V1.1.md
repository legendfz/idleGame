## 1. 六级品质配色与特效体系

### 1.2 品质在各场景中的渲染规则

#### 物品名渲染
```
{前缀}{物品名}
```
- 前缀 + 物品名同色（品质主色）
- 凡品/良品：normal weight
- 上品及以上：`font-weight: 600`
- 仙品/神品：额外 `text-shadow: 0 0 6px {主色, 0.4}`

#### 背包列表行
```
┌─────────────────────────────────┐  ← border-left: 3px {品质色}
│ {emoji} {前缀}{名称}    +{lv}  │     背景: 品质衬色
│   {属性}  {标签}               │
└─────────────────────────────────┘
```
- 左边框 3px 品质色
- 行背景用对应品质衬色（微弱底色，区分层次）

#### 装备详情弹窗
- 弹窗 border: `1px solid {品质色}`
- 极品★及以上：border 宽度 2px
- 仙品✦：弹窗顶部追加 `✦ 仙品 ✦` 标签条，橙色底
- 神品✧：弹窗顶部追加 `✧ 神品 ✧` 标签条，红色底 + 脉动动画

#### 战斗日志掉落行
```
凡品/良品: → 📦 获得 ○木棍
上品:      → 📦 获得 ◆降妖杖 ✨
极品:      → 📦 获得 ★铁棒 ✨✨
仙品:      → 📦 获得 ✦三叉戟 ✨✨✨  （整行橙色）
神品:      → 📦 获得 ✧如意金箍棒 ✨✨✨✨✨  （整行红色+加粗）
```

### 2.3 显示逻辑

**阶段 1 — 弹窗出现（0~300ms）**
```
遮罩 opacity 0 → 0.7
弹窗 scale(0.9) → scale(1) + opacity 0 → 1
```

**阶段 2 — 数字递增（300~1500ms）**
```
🪙 灵石   0 → 128,400    (countUp, 800ms)
✨ 经验   0 → 15,600     (countUp, 800ms, 延迟 100ms)
🍑 蟠桃   0 → 2          (countUp, 400ms, 延迟 200ms)
```

**阶段 3 — 战斗成果淡入（1500~2000ms）**
```
战斗成果区域 opacity 0 → 1，slide-up 8px
```

**阶段 4 — 装备掉落逐行出现（2000~2500ms）**
```
每件装备从左 slide-in，间隔 200ms
品质高的装备行有品质色左边框
```

**阶段 5 — 按钮出现（2500ms）**
```
[ ✅ 全部领取 ] 按钮 opacity 0 → 1
```

### 2.4 区块规范

#### 基础收益区
```
┌─ 基础收益 ─────────┐
│ {ico} {名称}   +{值} │
└────────────────────┘
```
- 名称左对齐，数值右对齐
- 数值用 `--font-mono` 等宽
- 数值色 = `--c-accent` 金色

#### 战斗成果区
- 仅当离线时间 > 5 分钟显示
- 如果升级了，等级变化用 `→` 连接，绿色

#### 装备掉落区
- 仅当有装备掉落时显示
- 每行格式：`{品质前缀}{名称}    {主属性}`
- 文字色 = 品质色
- 如果掉落了仙品/神品，该行追加 ✨ 闪烁

### 2.5 边界情况

| 场景 | 显示 |
|------|------|
| 离线 < 1 分钟 | 不弹窗 |
| 离线 1~5 分钟 | 仅基础收益，无战斗成果 |
| 离线 > 24 小时 | 按 24h 封顶计算，显示「最多计算 24 小时」 |
| 无装备掉落 | 隐藏装备掉落区 |
| 灵石/经验为 0 | 不会出现（有挂机收益公式保底） |

### 2.6 领取按钮

```css
.offline-claim-btn {
  background: var(--accent);
  color: var(--bg);
  border: none;
  padding: 12px 48px;
  border-radius: 8px;
```

### 3.7 MVP 简化方案

如果后端暂未就绪，反馈可以：
1. 存入 `localStorage`，后续版本上传
2. 或生成 `mailto:` 链接发送邮件
3. 或提交到 Google Form（无后端依赖）

```ts
// MVP: Google Form 方案
function submitFeedback(type: string, text: string, context: object) {
  const formUrl = 'https://docs.google.com/forms/d/e/{FORM_ID}/formResponse';
  const params = new URLSearchParams({
    'entry.TYPE_FIELD': type,
    'entry.TEXT_FIELD': text,
    'entry.CONTEXT_FIELD': JSON.stringify(context),
  });
  // 使用 fetch + no-cors 或 iframe 提交
  fetch(`${formUrl}?${params}`, { method: 'POST', mode: 'no-cors' });
}
```
