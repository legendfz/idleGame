# UI 设计规范 — 地下城堡风格（Dungeon Survivor Style）

> 版本：1.0 | 作者：CDO | 日期：2026-03-02
> 主题：西游记 × 暗黑地牢

---

## 1. 视觉风格定义

### 核心风格
- **暗黑地牢 + 西游记**：保留西游记世界观与角色，视觉从"水墨国风"转为"中世纪地牢"质感
- 参考：《地下城堡》《地下城堡2》《Slay the Spire》《暗黑地牢》

### 风格关键词
- 暗色系、高信息密度、卡片列表、节点地图、面板分区、纹理边框

## 3. 排版与字体

- **主字体**：`'Segoe UI', system-ui, sans-serif`（西文）
- **中文**：系统默认
- **等宽**：`'Cascadia Mono', 'Consolas', monospace`（数值/日志）
- **标题**：14-16px，暗金色 `#c9a84c`，`font-weight: 700`
- **正文**：13px，灰白 `#d4cfc4`
- **辅助**：11-12px，暗灰 `#7a7466`
- **行高**：1.5

---

## 4. 卡片与面板

### 卡片样式
```css
.card {
  background: #201c14;
  border: 1px solid #3d3422;
  border-radius: 4px;  /* 地牢风格偏硬朗，小圆角 */
  padding: 12px 16px;
  box-shadow: 
    inset 0 1px 0 rgba(201, 168, 76, 0.05),  /* 内发光：微弱金边 */
    0 2px 8px rgba(0, 0, 0, 0.5);             /* 外阴影：深沉 */
}
```

### 面板分区
- 属性面板、装备面板、技能面板各自独立卡片
- 卡片标题左侧加 2px 暗金竖线装饰
- 面板间距 8px

### 列表项
- 高信息密度，单行显示关键数据
- 左侧状态图标 + 名称 + 右侧数值
- hover/active 边框变为 `#6b5a2f`

---

## 5. 地图探索（节点连线）

### 样式参考
- **Slay the Spire 路线图**风格
- 垂直布局，从下到上
- 节点：圆形/六边形，40px，带边框
- 连线：1px 暗金虚线
- 当前节点：发光效果（box-shadow 暗金光晕）
- 已通过：实心，暗绿边框
- 未解锁：暗灰半透明
- Boss 节点：更大，暗红边框 + 脉冲动画

### 节点类型图标
- ⚔️ 战斗（普通）
- 💀 Boss
- 🏕️ 休息/商店
- ❓ 未知事件

---

## 6. 战斗界面

- 背景：深黑渐变 `linear-gradient(180deg, #0d0d0d 0%, #1a1710 100%)`
- 敌人区域：暗红阴影氛围 `box-shadow: 0 0 40px rgba(139, 32, 32, 0.3)`
- HP 条：暗红填充 `#8b2020`，背景 `#2a2318`
- 伤害飘字：暗金（暴击）/ 灰白（普通）
- Boss 出场：全屏暗红闪烁 + 震动效果
- 战斗日志：等宽字体，每行前加时间戳灰色

---

## 7. 排行榜 / 成就 / 背包

### 排行榜
- 列表式，每行一个记录
- 排名 1-3 用金/银/铜色图标
- 当前玩家行高亮（暗金边框）
- 数值右对齐，等宽字体

### 成就
- 卡片列表，已完成=暗绿底 + ✓
- 未完成=暗灰底，进度条显示
- 成就名称 14px 粗体，描述 12px 灰色

### 背包
- 网格/列表可切换
- 物品卡片左边框用品质色
- 装备详情弹窗：暗色面板 + 暗金边框
- 分解/强化按钮：暗红/暗金

---

## 8. 底部导航

## 10. 动效规范

- **过渡**：150ms ease，不用 spring/bounce（保持沉稳）
- **卡片展开**：高度过渡 200ms
- **节点地图滚动**：平滑滚动
- **Boss 出场**：0.5s 暗红闪烁 + 缩放
- **伤害数字**：向上飘 800ms 淡出
- **禁用花哨动画**：不用彩虹、粒子等——保持暗沉克制

---

## 11. 西游记元素保留

虽然视觉风格转暗，但保留：
- 角色名称（悟空/八戒/沙僧/唐僧）
- 地点名称（花果山/火焰山等）
- 境界体系（炼气/筑基等修仙术语）
- 装备名称（金箍棒/九齿钉耙等）
- 文案语气保持西游记世界观

只是**视觉呈现**从明亮国风 → 暗黑地牢。
