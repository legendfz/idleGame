## 1. 伤害数字飘字系统

### 1.1 飘字类型与样式

| 类型 | 格式 | 颜色 | 字号 | 动画 |
|------|------|------|------|------|
| 普通伤害 | `-{n}` | `#f44336` 红 | 14px | 上飘淡出 |
| 暴击伤害 | `-{n} 💥` | `#ff5722` 深红 | 18px 加粗 | 放大→上飘 |
| 技能伤害 | `-{n} 🔥` | `#ff9800` 橙 | 16px | 上飘 + 横向扩散 |
| 治疗 | `+{n}` | `#4caf50` 绿 | 14px | 上飘淡出 |
| 格挡/减伤 | `-{n} 🛡️` | `#9e9e9e` 灰 | 12px | 缓慢上飘 |
| 闪避 | `MISS` | `#8b8b9e` 灰 | 12px 斜体 | 横向滑过 |
| 反伤 | `↩ -{n}` | `#9c27b0` 紫 | 14px | 反向下飘 |

### 1.2 飘字动画规范

**普通伤害飘字**：
```css
@keyframes dmg-float {
  0%   { transform: translateY(0); opacity: 1; }
  30%  { transform: translateY(-12px); opacity: 1; }
  100% { transform: translateY(-30px); opacity: 0; }
}
/* duration: 800ms, ease-out */
```

**暴击飘字**：
```css
@keyframes dmg-crit {
  0%   { transform: translateY(0) scale(1.8); opacity: 1; }
  15%  { transform: translateY(-5px) scale(1.0); opacity: 1; }
  30%  { transform: translateY(-12px) scale(1.0); opacity: 1; }
  100% { transform: translateY(-40px) scale(1.0); opacity: 0; }
}
/* duration: 1000ms, ease-out */
/* 初始放大 1.8x → 迅速缩回 → 上飘消失 */
```

**闪避飘字**：
```css
@keyframes dmg-miss {
  0%   { transform: translateX(-20px); opacity: 0; }
  20%  { transform: translateX(0); opacity: 1; }
  80%  { transform: translateX(0); opacity: 1; }
  100% { transform: translateX(20px); opacity: 0; }
}
/* duration: 600ms, ease-in-out */
```

### 1.3 飘字位置

```
  ┌──────────────────────────┐
  │ 👹 石头精         [3/10] │
  │ ❤️ ████░░░░░░  420/1,200 │
  │              -156         │ ← 飘字从血条右侧起始
  │            -312 💥        │ ← 多个飘字错开 Y 轴
  └──────────────────────────┘
```

- 飘字起始位置：血条数字右侧
- 多个飘字同时出现时，Y 轴随机偏移 ±8px 避免重叠
- 同一目标最多同时显示 3 个飘字，超出排队

### 1.4 数字格式化

| 数值范围 | 显示 | 示例 |
|----------|------|------|
| < 1,000 | 原值 | `-847` |
| 1,000 ~ 9,999 | 千分位 | `-1,204` |
| 10,000 ~ 99,999 | 万 | `-1.2万` |
| 100,000 ~ 999,999 | 万 | `-12.4万` |
| ≥ 1,000,000 | 百万 | `-124万` |

## 2. 暴击特效系统

### 2.1 暴击触发视觉序列

```
时间线：
0ms     100ms   200ms   400ms        1000ms
│────────│───────│───────│────────────│
│ 闪屏   │ 震动  │飘字弹出│ 飘字上飘    │
│        │       │       │  淡出消失   │
```

**阶段 1 — 屏幕闪光（0~100ms）**：
```
整个 ContentArea 叠加一层白色蒙版
opacity: 0 → 0.15 → 0（100ms）
仅暴击时触发，普通攻击无此效果
```

**阶段 2 — 敌方面板震动（100~200ms）**：
```css
@keyframes panel-shake {
  0%, 100% { transform: translateX(0); }
  25%      { transform: translateX(-3px); }
  50%      { transform: translateX(3px); }
  75%      { transform: translateX(-2px); }
}
/* duration: 100ms, 仅 EnemyPanel 元素 */
```

**阶段 3 — 暴击飘字（200~1000ms）**：
```
  -2,408 💥
  ↑ 放大 1.8x → 缩回 1.0x → 上飘消失
  字号 18px，加粗，深红色
```

### 2.2 暴击战斗日志增强

普通攻击日志：
```
🐒 悟空 → 👹 石头精  -1,204
```

暴击日志（差异对比）：
```
🐒 悟空 →→ 👹 石头精  -2,408 💥 暴击！
```

差异：
- `→` 变为 `→→`（双箭头）
- 数字加粗
- 追加 `💥 暴击！`
- 整行背景色：`rgba(244, 67, 54, 0.08)` 微红底

### 2.3 连续暴击特效

连续暴击 2 次及以上时的额外反馈：

| 连击数 | 额外效果 |
|--------|---------|
| 2 连暴 | 日志追加 `🔥 二连暴击！` |
| 3 连暴 | 日志追加 `🔥🔥 三连暴击！`，边框闪金 |
| 5+ 连暴 | 日志追加 `💥💥💥 暴击风暴！`，全屏金色闪屏 |

## 3. 技能释放动画系统

### 3.1 技能释放序列

```
时间线：
0ms        300ms      500ms      800ms       1200ms
│──────────│──────────│──────────│───────────│
│ 技能名横幅│ 蓄力效果 │ 释放效果  │ 伤害飘字   │
│ slide-in │ 脉动     │ 扩散     │ 结算      │
```

### 3.2 技能名横幅动画

```
阶段：技能名居中横幅（0~300ms）
─────────────────────────────

         ──── 七十二变 ────

从左侧 translateX(-100%) 滑入
到达中央后停留 200ms
然后 opacity 淡出

横幅样式：
- 文字：--c-accent 金色，16px 加粗
- 左右装饰线：── ──
- 背景：transparent
```

### 3.3 技能类型与特效

#### 单体技能

```
🐒 ──── 棒打三千 ────
   如意棒连击！→ 👹 -3,500 🔥

效果：
- 飘字出现 3 次（模拟连击）
  -1,200 → -1,200 → -1,100
  间隔 150ms，位置依次上移
- 最后汇总显示总伤害
```

#### AOE 技能

```
🐒 ──── 大闹天宫 ────
   金箍棒横扫！全体 🔥
   👹 石头精  -2,100
   👹 树妖    -2,100
   👺 赤尻马猴 -1,800

效果：
- 所有敌人血条同时闪烁
- 每个敌人各自出飘字（同时触发，无延迟）
- EnemyPanel 整体震动一次
```

#### 治疗/辅助技能

```
🧘 ──── 大悲咒 ────
   梵音治愈，全体恢复！
   🐒 悟空  +800 ❤️
   🐷 八戒  +800 ❤️

效果：
- 绿色 + 号飘字（向上）
- 血条从旧值平滑增长到新值
- 无震动效果
```

#### Buff/Debuff 技能

```
🐒 ──── 火眼金睛 ────
   看破虚妄！暴击率 +15%
   ⏱️ 持续 30秒

效果：
- buff 显示在角色名后：🐒 悟空 [👁️]
- 日志中用方括号标记 buff 名
- buff 到期时日志提示：[👁️ 火眼金睛 已消失]
```

### 3.4 Buff/Debuff 状态栏

角色有 buff 时，在属性速览行显示：

```
  ⚡328  ❤️2,800  ✨3.4K/5.0K
  [👁️ 28s] [💪 15s]              ← buff 图标 + 剩余秒数
```

敌人有 debuff 时，在敌人名后显示：

```
  👹 石头精 [🐌]  ❤️ ████░░░░░░
```

| Buff/Debuff | 图标 | 说明 |
|-------------|------|------|
| 攻击提升 | 💪 | 红色 |
| 防御提升 | 🛡️ | 蓝色 |
| 暴击提升 | 👁️ | 金色 |
| 速度提升 | 💨 | 白色 |
| 减速 | 🐌 | 灰色 |
| 中毒 | 🟢 | 绿色 |
| 灼烧 | 🔥 | 橙色 |
| 眩晕 | 💫 | 黄色 |

## 4. Boss 战特效

### 4.1 Boss 出场序列

```
时间线：
0ms          500ms        1000ms       2000ms       2500ms
│────────────│────────────│────────────│────────────│
│ 屏幕变暗    │ Boss框滑入 │ 台词打字   │ 框收缩     │ 战斗开始
│ 遮罩0→0.3  │ scale弹入  │ 逐字显示   │ 融入面板   │
```

**阶段 1 — 暗幕（0~500ms）**：
```
ContentArea 叠加黑色遮罩
opacity: 0 → 0.3
同时 EnemyPanel 和 BattleLog 暂停更新
```

**阶段 2 — Boss 框弹入（500~1000ms）**：
```
  ═══════════════════════════
    🐉 BOSS 出现！
    「混世魔王」
    ❤️ ██████████  50,000
  ═══════════════════════════

  动画：从 scale(0.5) + opacity(0)
       到 scale(1.05)（弹性过冲）
       到 scale(1.0)
  duration: 500ms
  timing: cubic-bezier(0.34, 1.56, 0.64, 1)
```

**阶段 3 — Boss 台词（1000~2000ms）**：
```
  ═══════════════════════════
    🐉 BOSS 出现！
    「混世魔王」
    ❤️ ██████████  50,000

    "花|果|山|是|我|的|地|盘|！"
  ═══════════════════════════

  台词逐字出现，每字 80ms
  打字机效果，光标 ▌ 闪烁
```

**阶段 4 — 过渡到战斗（2000~2500ms）**：
```
  Boss 框收缩融入 EnemyPanel
  遮罩 opacity 0.3 → 0
  BattleLog 恢复滚动
  Boss 血条替换普通怪物面板
```

### 4.2 Boss 血条特殊样式

```
  🐉 「混世魔王」       [BOSS]
  ❤️ ████████░░  40,000/50,000
  ──────────────────────────
```

与普通怪的区别：
- 名称用「」包裹，金色 `--c-accent`
- 右侧 `[BOSS]` 标签，红色
- 血条下方额外分隔线
- 血条颜色固定红色（不随百分比变色）
- 名称文字加粗

### 4.3 Boss 击杀特效

```
时间线：
0ms        300ms      800ms       1500ms      2500ms
│──────────│──────────│───────────│───────────│
│ 血条归零  │ 击杀动画 │ 战利品弹窗│ 弹窗展示  │ 关闭
│ 最后一击  │ 爆炸碎裂 │ 弹入      │ 可交互    │
```

**击杀动画**：
```
  🐉 「混世魔王」
  ❤️ ░░░░░░░░░░  0/50,000
        💥💥💥
      💀 击 败 ！
        ✨✨✨

  Boss 行闪烁 3 次（opacity 1→0→1，每次 100ms）
  然后 "碎裂" 效果：行内文字随机偏移 + 淡出
  💀 击败！文字 scale 弹入（居中大字）
```

**战利品弹窗（多物品）**：
```
  ╔═══════════════════════════╗
  ║  🏆 Boss 击败！            ║
  ║  「混世魔王」              ║
  ║  ═════════════════════    ║
  ║                            ║
  ║  战利品：                  ║
  ║  ⚔️ ★如意金箍棒   ⚡+180  ║ ← 逐行出现
  ║  💊 ●仙丹 ×3     ❤️+500  ║    间隔 300ms
  ║  🪙 +5,000                ║
  ║  ✨ +1,200                ║
  ║                            ║
  ║       [ ✅ 领取 ]         ║
  ╚═══════════════════════════╝

  动画：弹窗整体 scale 弹入
  战利品逐行从左 slide-in，间隔 300ms
  品质高的物品行闪烁金边
```

## 6. 波次过渡动画

### 6.1 普通波次切换

```
  👹 树妖 💀 击败！
    → 🪙 +300  ✨ +80

  ── ── ── ── ── ── ──

       第 4 波

  ── ── ── ── ── ── ──

  👹 花妖      ❤️ ██████████
  👹 石怪      ❤️ ██████████
```

- 虚线分隔 + 波次号居中

### 6.2 Boss 波次切换

## 7. 挂机状态视觉

### 7.1 挂机收益区脉动

```
  💰 +350/秒  ✨ +42/秒
  ⏱️ 挂机中… 02:34:12
```

- `⏱️ 挂机中…` 的 `…` 循环动画（1 点→2 点→3 点→1 点，间隔 500ms）
- 收益数字每秒微弱 pulse（scale 1.0 → 1.03 → 1.0）
- 计时器每秒更新

### 7.2 后台挂机指示

App 在后台时，顶栏资源数字标记为估算值：

```
🪙 ~15,200  💠 38  ✨ Lv.15
      ↑ 波浪号前缀表示估算
```

回到前台时去掉 `~`，countUp 到实际值。

## 8. 战斗日志格式汇总

| 事件 | 日志格式 | 特殊样式 |
|------|----------|---------|
| 普通攻击 | `{atk} → {def} -{dmg}` | 无 |
| 暴击 | `{atk} →→ {def} -{dmg} 💥 暴击！` | 加粗，微红底 |
| 技能 | `{atk} ──── {技能名} ────`<br>`   {描述} -{dmg} 🔥` | 金色技能名 |
| 治疗 | `{src} → {tgt} +{heal} ❤️` | 绿色数字 |
| 格挡 | `{def} 🛡️ 格挡！ -{reduced}` | 灰色 |
| 闪避 | `{def} 💨 闪避！` | 灰色斜体 |
| 反伤 | `{def} ↩ 反伤！→ {atk} -{dmg}` | 紫色 |
| 击杀 | `{mob} 💀 击败！`<br>`  → 🪙 +{g} ✨ +{e}` | 无 |
| 升级 | 双线框独立块 | 金色 |
| Boss出场 | 等号框独立块 | 金色，全屏特效 |
| 角色倒下 | `💀 {name} 倒下了！` | 红底 |
| Buff | `[{ico} {name}] 效果生效/消失` | 对应颜色 |
| 波次 | `── 第 {n} 波 ──` | 居中 |
